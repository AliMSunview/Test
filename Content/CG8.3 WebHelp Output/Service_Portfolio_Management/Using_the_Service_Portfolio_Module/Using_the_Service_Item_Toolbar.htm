Using the Service Item Toolbar

The Service Item Toolbar lets you perform common operations by clicking command buttons in the form of icons.

Icon

Description

Saves the service item.

Discards any changes made to the service item.

Creates a new service item. When used within the service item form, it creates a new service item in the same window.

Creates a duplicate service item. When used within the service item form, it creates a duplicate service item in the same window.

Deletes the service item.

Prints the service item.

E-mails the service item.

Displays a link to the service item.
