Overview of the View Editor

The View Editor allows Administrators to create and edit system views from within a ChangeGear module, (e.g., incident, problem, change, CMDB, knowledge base). The create and edit functions are offered in addition to the View Editor function in the ChangeGear Desktop Client. Views created using the Desktop Client can be managed by Administrators from the ChangeGear View Editor on the web.

Note: To enable the View Editor, a user's role must have the Administration > Edit privilege enabled. Typically, administrators are the only ones who can edit system views.
