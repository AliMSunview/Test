Creating and Editing Bubble Charts

Report Designer, you can create bubble charts which are diagrams that display a three-dimensional data set. Bubble charts are similar to scatter charts, except they also compare the relative size of each data point in addition to its distribution. You can specify the columns that make up each data set and apply filtering to further refine the scope of data represented.

To create a bubble chart report:

  1. Click Reports on the primary ChangeGear navigation menu.

  2. Click the New icon in the tool bar.

  3. The Report Creation dialog displays. Select the Bubble Chart report.

  4. Click Next. The Bubble Chart Creation screen displays.

  5. Select a module to source the data from.

  6. Select a view that will provide the data available in the report. The report will use the same filter criteria as the view.

  7. Enter a Name for the new report.

  8. Optionally, enter a Description of the report.

  9. Select a Report Category from the list. The Report Category determines which category within the module to save the report. The My Reports category is specific to your user account and is not visible to other users.

Data Tab

The Data tab allows you to define which fields to use to populate the bubble chart. The X-Axis field lets you specify which column to use for the horizontal axis, and the Y-Axis field lets you specify which column to use for the vertical axis. For example, suppose you want to view the number of incidents of a specific status and priority. You would set the X-Axis field to Status and the Y-Axis field to Priority. The result is a chart of the number of incidents matching both column values, with each bubble sized to the number of incidents in each match. Criteria Filter Tab

The Criteria Filter tab lets you provide additional data filtering parameters.

To apply a filter:

  1. Select an axis to filter.

  2. Select a matching condition using the radio buttons.

    • Match All Conditions means the data must meet each one of the filtering options specified to appear in the view result set, (i.e., acts like an AND condition for filtering options).

    • Match Any Condition means the data can match any one of the conditions and is displayed in the view result set, (i.e., acts like an OR condition for filtering options).

  3. Use the field drop-down to select a field to match on.

  4. Use the value drop-down to select or provide a value to match on. The items available in the drop-down change depending on the field selected.

  5. Press Add to add more filtering criteria. Press to delete criteria. You can add multiple filtering criteria as necessary for your report.

Create the Report

When you are finished entering your criteria, click Create to create the report.

Note: You can create a new report from an existing report by modifying the existing report and clicking Save As and providing a different name.

Editing a Bubble Chart Report

ChangeGear allows you to edit the description, data columns, and filter criteria of existing bubble chart reports.

To edit a bubble chart report:

  1. Click Reports on the primary ChangeGear navigation menu.

  2. Select a Bubble Chart report from the Reports grid.

  3. Click the Edit icon in the tool bar.

  4. Edit the Description as necessary.

  5. Edit the Data Tab and Criteria Filter options as shown above in the Create a Report section.

  6. Click Save.

    Note: You can create a new report from the current report by clicking Save As.
