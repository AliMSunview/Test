Overview of the Report Designer

The Report Designer on the web allows you to create, edit, and delete various types of reports and charts from within the ChangeGear Reports module. This web based report editing is in addition to the Report Designer functions in the ChangeGear Desktop Client.

A variety of report types are available for user consumption. Click on each report type for more details.

  • Table

  • Bar chart

  • Gauge

  • Pie chart

  • Score card

  • Line chart

  • Donut Chart

  • Bubble Chart

Note:  You may not have access to this ChangeGear feature depending on your permissions or ChangeGear edition. Contact your ChangeGear Administrator for privilege/permission changes.

Reports and Dashboard Widgets

Charts, gauges, and score cards that are created in the Report Designer on the web can be added to the ChangeGear Dashboard. Once a chart is created, the chart can be displayed in the Dashboard or Workspace by creating the appropriate Workspace or Dashboard widget and applying the chart to that widget. For example, you can create a Bar Chart widget in the Dashboard which references a Bar Chart report created in the Report Designer on the web.

Note:  Table reports cannot be used as widgets.
