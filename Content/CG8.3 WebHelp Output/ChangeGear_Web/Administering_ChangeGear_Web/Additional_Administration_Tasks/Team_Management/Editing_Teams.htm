Editing Teams

The Team Manager module enables you to edit team information, add members to a team, and remove members from a team.

To edit team information:

  1. Navigate to a module, (e.g., incident, problem, change request), from the ChangeGear primary navigation menu.

  2. Select Team Manager from the Other category.

  3. Select a team from the list.

  4. Click the Edit icon in the Team tool bar.

  5. The Team Information dialog displays with the Team Info tab highlighted. You can revise the Name, Description, CC E-mail, and Team Owner fields as necessary. At this point, you can revise team information or navigate to the Members tab to add or remove team members.

To add members:

  1. Navigate to the Members Tab to view all team members.

  2. To add a member to the team, click Add . The Select Group Members dialog displays.

  3. You can either scroll through the list to select a member name, or you can search for a name by entering a search string and pressing Find.

    Note: You may include other fields not displayed in the Select Group Member dialog using the Field Chooser. For example, you could add the e-mail field to the dialog display to view e-mail addresses. You can group members using the Field Chooser, for instance, by Location Name. Please refer to Adding and Removing Columns in the Using the Grid section for a complete explanation.

  4. Select the member and scroll to the bottom of the dialog. Click OK.

  5. Click the Save icon to save the changes. Close the dialog.

To remove members:

  1. Navigate to the Members Tab to view all team members.

  2. To remove a member of the team, select a team member from the list and click Remove .

  3. Click the Save icon to save the changes. Close the dialog.
