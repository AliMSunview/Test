Deleting SLAs

You should refrain from deleting SLAs from ChangeGear as this negates tracking and reporting. If you no longer want the SLA to be active, deactivate the SLA. See Activating SLAs for more information.

If you wish for the SLA to no longer be applied to new tickets, change the conditions so that incoming tickets will no longer match. For example, you could set the condition to [Item ID] = 0.

To delete an SLA:

  1. Open the SLA Manager.

  2. Select the SLA you wish to delete.

  3. Click Delete. A confirmation dialog box displays.

  4. Choose Yes to delete the SLA.

Note: Any tickets that the SLA is applied to will continue to display the state they were in when the SLA was in effect. This also applies to tickets that were closed while the SLA was in effect.
