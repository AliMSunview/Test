Adding and Editing Change Windows

ChangeGear web administration enables you to create and edit Change Windows if you are licensed for this feature.

Note: Change Window and Blackout Period features require the Advanced Change Management license. This includes any Business Policy Automations (BPAs), system rules, and management features pertaining to the Change Window and Blackout Period features.

1.  Click Administration > Modules > Change > Change Windows. (If you do not have the needed license, this option does not appear.) ChangeGear shows the Change Windows screen.

2.  To add a Change Window, click the plus sign to open the slide-out editor.

-OR-

      To edit a Change Window, hover over the existing window in the grid to display the edit () and delete ( ) icons, click to display the corresponding slide-out editor, and make changes to settings/ information as described in the steps below.

3.  Click on the Enabled option () to enable/disable the Change Window you are creating/editing.
 

4.  Enter the new Change Window's Name, Description, Start, and End dates. You can select the ’All day’ option if you don’t want to include a specific time range.

5.  Make the window recurring, if appropriate (see Making Change Windows and/or Blackout Periods Recurring as shown below).

6.  Review any tickets that will be affected by this new/edited Change Window (see Viewing Impacted Tickets, as shown below).

Note: If you disable the Allow all tickets to be scheduled during this period option so that certain tickets cannot be scheduled during the set time period, you must define the conditions restriction. Furthermore, you must follow the condition format as follows: ([PriorityName] = X). For example, if you enter the condition as ([Urgent] = ‘1 – High’, only tickets with a designation of High priority will be allowed to be scheduled during the set time period. Only when the condition(s) is formatted properly, can you save it.

7.  Click the save icon to save the Change Window.

8.  Click X to close the slide-out editor and return to the grid.
