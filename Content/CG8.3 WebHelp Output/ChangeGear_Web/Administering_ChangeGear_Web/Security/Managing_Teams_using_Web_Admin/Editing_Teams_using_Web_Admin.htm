Editing Teams Using Web Admin

After a team is created, or during the creation, you can select the team members, modify team information, change team owners, enter a new team e-mail address, or change guest roles. Members can be added or deleted as necessary and have their role within a security team modified to adjust for differing responsibilities or employment shifts.

After creating a team, you cannot modify the type, i.e., you cannot change a notification team to a security team.

To view how to modify team membership, see Modifying Team Membership.

To modify Team information:

  1. From the Web Administration page, click Security> Teams. ChangeGear displays the Teams page.

  2. To edit an existing team, hover over the team's row and click the pencil icon. ChangeGear opens that row in the grid so you can edit the team.

  1. Make any necessary changes.

  2. Click OK to save your changes.
