Adding Teams Using Web Admin

To create a Team:

  1. From the Web Administration page, click Security> Teams. ChangeGear displays the Teams page.

  1. To add a new team, click the plus sign in the upper right corner. ChangeGear opens the Team slide-out editor.

          

  1. In the Organization Name field, choose the name of the organization or Global. Note: This field is visable only if you have an MSP license.

  2. Type the team name in the Name field.

    Tip: Include the word "Team" in the name so you can find your teams more easily in ticket forms.

  3. In the Description field, identify it as a notification team or security team.

Note: Teams may not be members of other teams.

  1. Next, select a Team Owner using the down arrow.

  2. In Cc: E-mail enter the email address used to notify this team. 

  3. In Type, click the Notification or Security button.

  • For Notification groups, you can specify another group to receive the notifications as well. Check Also notify a group defined in your corporate directory if you want to include a group defined in your Active Directory (AD) to this team's notification. Then enter the group email address in External Group. For more information on Notification Teams, see Notification Teams with Web Admin.

  • For Security groups, you can enter a Guest Role. This is the role that non-members will have when they view your tickets. If you do not want non-team members to have any privileges, select No Privileges from the list of available roles. For more information on Security Teams, see Security Teams using Web Admin.

  1. Click Save  to save your changes.

Adding Members to a Team

  1. Click on the Members tab to identify members for the team.

          

  1. Click  to add a member to this team. ChangeGear lists the available members. Click next to the name you want to add.

          

  1. Click Save  to save your changes. Continue adding members as needed, in the same manner.

Viewing System Information about a Team

When you create a new team and save it, ChangeGear automatically assigns system information that you can view.

  1. Create a new team and save the settings.
  2. On the main screen, highlight the new team and click on the  to edit that team. ChangeGear opens the team slide-out editor.
  3. Click on the System tab.

          

  1. On this tab you can view the OID, Created by name, Created date, Modified by last, Modified last date, and Accessed by.
