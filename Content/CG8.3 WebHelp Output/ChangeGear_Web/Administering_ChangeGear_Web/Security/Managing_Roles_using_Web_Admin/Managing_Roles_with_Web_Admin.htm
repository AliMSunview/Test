Managing Roles with Web Admin

The Web Admin module lets you manage the creation, editing, and removal of roles in ChangeGear. From the Roles page you can manage your roles and define the privileges and the groups who will receive communications, such as notifications and e-mails, when these roles are utilized.

If you decide to use security teams, you may wish to create team-specific roles to use.

You can manage your users from the Users page to create and edit users.

You can manage teams from the Teams page to create and edit teams.
