Defining the User's Role in Web Admin

When you are creating or editing Users, you can define the role requester when completing the User Profile.

To define the Role for the User:

  1. From the Web Administration page, click Security> Users. ChangeGear displays the Users page.

  1. Hover on the user's row, then click the pencil icon.

  2. ChangeGear opens the slide-out editor. In the User Profile section, click the Role drop-down menu.

  3. Select the role from the list.

  4. Complete the remaining fields in the dialog.

  5. Click OK.
