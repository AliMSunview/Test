Administering Multi-Tenant Organizations

Administering ChangeGear in multi-tenant organization mode is similar to administering a normal installation of ChangeGear. The key to remember is to create organizations first and then set visibility for lookup lists and other items. We recommend that you set up ChangeGear in the following steps:

  1. Create Organizations
  2. Create Teams
  3. Create Departments and Locations
  4. Create Users
  5. Define Lookup Lists, Notifications, Views, and BPAs
  6. Import CIs.
