Creating Organizations

Organizations are the keystone of an MSP implementation. Create organizations before any other configuration change as organizations control the visibility of items to users.

To create organizations:

  1. From the Web Administration page, click Security>Organizations. ChangeGear displays the Organizations page.

           

  1. To add a new organization, click  in the upper right corner. ChangeGear opens the Organization slide-out editor. Fro

           

Note: While creating organizations, you can also add specific users—the business contact, the technical contact, and the approver—by first creating them in the Add User page.

  1. Enter the Name of the organization, which corresponds with the customer’s name.

  2. Enter the Description of the organization.

  3. Select the Business Contact name from the drop-down and enter the associated Phone Number.

  4. Select the Technical Contact, and enter the Domain Name.

  5. Select the Services Approver name and enter their E-mail Address.

  6. To enable the Organization you just set up, slide the Enabled bar in the upper right corner to the right.

  7. Finally, click Save .
