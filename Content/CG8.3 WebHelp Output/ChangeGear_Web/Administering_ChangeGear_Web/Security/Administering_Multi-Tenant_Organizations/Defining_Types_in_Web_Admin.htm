Defining Types in Web Admin

Type lookup lists, such as Incident, Problem, or Change Types, are slightly different. These types of lists are hierarchical lookup lists, such that list items inherit certain attributes from a parent item. One of the attributes the lists inherit is the visibility of the root or parent node. When creating organization‐specific types, ensure that you select the parent node before creating a new list item.
