Editing a User with Web Admin

In the Users screen of the Web Administration module, you can edit your user information.

To edit Users:

  1. Navigate to Web Admin > Platform > Security > Users. ChangeGear displays the Users screen.

  2. Hover over the user you want to edit and click the pencil icon on the right.

  3. Enter your changes.

  4. Click the save icon .
