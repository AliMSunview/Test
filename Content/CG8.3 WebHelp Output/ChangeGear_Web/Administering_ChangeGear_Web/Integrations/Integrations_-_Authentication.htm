Authentication

Authentication information from Azure (Microsoft Teams) and/or Slack App can be optionally provided in this form for ChangeGear to connect and retrieve the user information associated with the user initiating the request from the external application.

1.  Go to the Authentication tab in the Connection area.

2.  Enter the appropriate authentication information.

3.  Click Test to make sure the information you have selected and entered is correct.

Note: It is also a good idea to test the overall configuration by creating and submitting a new ticket for the new (test) channel to make sure it appears in the external app.
