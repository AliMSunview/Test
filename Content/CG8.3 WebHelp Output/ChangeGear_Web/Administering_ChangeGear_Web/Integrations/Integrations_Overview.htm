Integrations

ChangeGear provides bi-directional data exchange with Microsoft Teams and Slack. Users can initiate creation of ChangeGear ticket requests (inbound communication) from within either of these applications. ChangeGear can also be setup to post updates (outbound communication) on the creation of tickets, ticket status etc. to channels inside Microsoft Teams or Slack. ChangeGear leverages Webhooks and connectors/bots that are made available by Microsoft Teams and Slack for data exchange.
