Setting Up Connections

Getting Started

1.  Navigate to the ChangeGear Web Admin and go to the "Integration" node and choose the application you want to integrate with as shown below:

Note: A ”Get Started” button only appears when setting up your first connection. Grid with the connections is displayed if at least one connection exists.








2.  Enter a unique but easy to recognize name (using alphanumeric characters only without spaces) and description for the integration to make it easier to identify as new integrations are added (e.g., Name: CreateTicketHelpDesk, Description: For creating Help Desk Incident tickets).

3.  Select the module (e.g., Incident, Problem, Change, etc.).

4.  Save () the connection.

Note: Make careful note of the exact Connection Name you added in ChangeGear as you need that information when setting up connections in the external apps.

Once you have completed the steps shown above, you need to set up Communications.
