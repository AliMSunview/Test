Overview of Chatbot Questions and Answer Models

Chatbots are intended for conversational questions and answers. Their intelligence comes from the training of intents which form what are known as models. ChangeGear uses Knowledge Base Text Similarity models to further enhance the intelligence workflow.

What is an intent?

An intent is what the user wants to know or do. Questions that can be asked by the user need to apply to a given intent. The result could be an answer or an action.

By using Intents, you can create chatbots that react to natural language and are able to engage in conversations with the users. The more intents that are provided with suitable questions/answers, the smarter the chatbot can appear. 

Basic principles for how to write good intent phrases: 

  1. Have a balanced number of questions per intent-- We recommend a minimum of 5 or 6 questions for each intent, but you can add as many as you like as long as all the intents have a balanced number of questions in each of them. The chatbot learns what you feed it, so the more intent questions you give it, the more precise it will be.
  2. Employ semantic diversity--Use different phrasings of a question when you create an intent.
  3. Create distinctive intents--Ensure that there is no overlap between questions in different intents. All questions that lead to the same answer should be grouped in the same intent.
  4. Use full sentences--Questions should be full sentences mimicking a conversation with a person. If you expect users to enter single words in the chatbot, you can create intents with questions containing the respective word and variations of it and advise in the answer that more information is required. This will encourage the user to reformulate the question and allow the chatbot to determine the respective intent to serve relevant information.

This example intent called 'Software Price' shows the different phrasings of a single question about the price/cost/charge for software. Note that it is assumed that similarly formed answers will be provided for each intent.

  • How much does the software cost?
  • What is the software price?
  • I want to know about the software price.
  • Where do I find software price information?
  • How much will you charge me?
  • Information about software price.
  • Where can I see your software price?
  • Where can I find the price for your software?

Intents

Each Chatbot Question & Answer model is made up of a set of intents.

  • Intent - An Intent is made up of a list of questions, but refers to the goal the customer has in mind when typing in a question or comment.
  • Question - The input from the user into the chatbot, either by statement or inquiry.
  • Answer - A random response, in the form of text, defined by an administrator.
  • Action - And action is a pre-defined response set up bu an administrator. An action can consist of specific text, or menu buttons with which the user interacts.
