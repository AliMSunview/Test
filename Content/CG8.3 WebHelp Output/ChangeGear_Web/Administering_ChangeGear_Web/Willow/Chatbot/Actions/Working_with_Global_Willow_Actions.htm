Working with Global Willow Actions

Actions are always global (shared). You can review and edit global actions (factory global actions along with any administrator-added global actions) that are available to all models within Willow. You can also delete an action from all intents using this screen. After you remove an action, ChangeGear reverts to presenting random answers (defined in the intent) in the chatbot.

  1. Click Willow > Chatbot > Actions. ChangeGear displays the Actions (Willow) screen. From here, you can add new actions, select columns to show in the grid, filter the grid, and refresh the page. You can also drill down on individual actions to edit or delete them.

  1. To edit an action, highlight its row, and click the pencil icon. ChangeGear displays the edit screen.

  1. For the selected action, you can now see its settings and change them as needed. You can:
    • Rename the action.
    • Add or change the action's description.
    • Change the action's type (DisplayMenu or TypeText to change the action's text). If you choose DisplayMenu ChangeGear changes the screen so you can define the menu items.

You can also delete the highlighted action by clicking on the trashcan icon on the far right.

  1. Once you have saved your changes, you need to retrain the model in order for your changes to take effect. See Implementing Changes to a Model.

Adding a New Action

  1. You can also add a new global action to a chatbot model. To do this, click on the + at the top right of the Actions (Willow) screen. ChangeGear opens the action menu.

  1. Enter the Name, Description and Status for the new action. Choose the action's type (DisplayMenu or TypeText to enter the action's text). If you choose DisplayMenu ChangeGear changes the screen so you can define the menu items. When you are done, click Save.

  1. Once you have saved your changes, you need to train the model in order for your changes to take effect. See Implementing Changes to a Model.
