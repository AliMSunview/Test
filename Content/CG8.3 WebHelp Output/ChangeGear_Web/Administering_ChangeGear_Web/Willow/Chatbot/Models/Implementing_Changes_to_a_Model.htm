Implementing Changes to a Model

Highlight a model on the Chatbot Models screen and click the Configuration icon  to configure an individual model. From the Model Configuration screen you can:

  1. Change the model. Make any changes you desire to the model or its components (intents, Q&A, actions).
  2. Train the model.  To update the Willow model, first you need to train the model to get your changes into a new Willow model. Until you perform a successful train, ChangeGear does not update the model with your recent changes.

    (Once you click Train the Model, periodically click Refresh Data to monitor the status and training results at the top left of the page.)

  1. Test this model.  You can test the model to make sure it is viable. Click Start Testing.  ChangeGear displays the Model Tester slide-out editor.

Enter the modified chatbot question you want to test, then click the Send arrow on the right.

If the model does not respond the way you intend, go back and edit it to correct any issue (see  Configuring a Chatbot Model for more details. Then save your changes, retrain the model, and test it again until you are satisfied.
