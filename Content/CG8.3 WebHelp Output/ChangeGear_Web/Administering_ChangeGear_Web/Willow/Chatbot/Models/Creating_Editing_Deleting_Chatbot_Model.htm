Creating/Editing/Deleting a Chatbot Model

You can create new models, as well as enable/disable existing models.

  • Enabling a model creates a Willow model. You then must train and test the Willow module.
  • Disabling a model deletes the Willow model from memory.
  • Deleting a model removes the corresponding Willow model.
  • Deleting a model removes its association with the Intents, but does not remove any Intents.

To create a new chatbot model:

  1. Go to the Willow > Chatbot > Models screen.
  2. Click on the + icon in the upper right.
  3. A new model line opens. Enter the Name for your new model, then click the Save icon.
  4. If you are creating multiple new models, instead click on the + icon to save and immediately start adding another model.

To enable/disable chatbot models:

  1. Go to the Willow > Chatbot > Models screen.
  2. Click on the Column Chooser icon in upper right corner. From the dropdown, drag and drop the Enabled column to the model list.
  3. Highlight the model you want to enable/disable. Click the pencil on the right to edit the model.
  4. In the Enabled column, click a blank checkbox to enable a model, or clear the checkbox to disable a model.
  5. Click on the save icon on the right to save your changes.

To delete a chatbot model:

  1. Go to the Willow > Chatbot > Models screen.
  2. Locate the model you want to delete on the list, select it, and click the Delete icon all the way over on the right side of the line.
  3. At the "Are you sure?" message click YES.
  4. ChangeGear removes the model you specified, deletes its association with any intents, but does not delete the intents. They remain available to use with other models.
