Configuring a Chatbot Model

ChangeGear Web Administration enables you to view and edit all aspects (intents, questions, actions) of a chatbot model from the Module Configuration screen.

NOTE: You must supply at least one response or action for each intent, otherwise you won't be able to train the model.

  1. In the Web Administration module, select Willow > Chatbot > Models. On this screen, you see a list of all your models.
  2. Hover over the row of the model you want to view or edit to select, and click on the Configure (gear) icon on the far right side of that row.  
  3. ChangeGear opens the Model Configuration screen, and the grid shows all intents associated with the model.

  1. In addition to the summary information along the top, this screen shows the intents associated with the selected model.
  2. To look at intent details, highlight an intent and click the Edit icon (pencil) in the action column on the far right side of that line. ChangeGear opens a slide-out editor on the right.

  1. You can now view the Name, Description, Status, Questions, and Responses assigned to that intent. You can also chose to Run an action for a specific question.  NOTE: You must supply at least one response or action for each intent, otherwise you won't be able to train the model.

    Note: The Edit Intent slide-out opens as read only, by default,.  To edit it, click on the pencil icon  top right, and then click Edit to confirm change to all models. ChangeGear then changes the pencil to a globe  to remind you that you are making changes to an intent used by other models as well.

  1. For each intent, you must assign a Response or Action for ChangeGear to perform when users are using the chatbot.
  2. For responses, click Show random answer from the list below.  To add a response, click the + in the upper right corner--ChangeGear opens a row for you to add a new response.

             

  1. For actions, click Run an action and the dropdown to select which action you want ChangeGear to run. Note: The actions available on this dropdown correspond to those defined in the Actions page.  See Working with Global Willow Actions for more details.

  1. If you delete an intent from a module, you can get different results based on where the intent is used:
    • If the intent is used only by this model and is not global, ChangeGear deletes the intent along with its questions and answers.
    • If the intent is shared with multiple models, ChangeGear removes the relationship with this model but does not delete the intent.
  2. If you make changes to any of these items, click the Save icon on the top right before closing the Edit Intent slide-out editor.
  3. You can also add a new intent to a chatbot model. To do this, click on the + at the top right of the Model Configuration screen. ChangeGear opens the in-line edit row and enables you to enter the Name, Description, Status, etc for the new intent. You can also enable or disable the intent. When you are done with this row, click the save new icon. 

 

  1. Once you have finished saving any changes made, you need to train the model to get a new Willow model and test it. See Implementing Changes to a Model.
