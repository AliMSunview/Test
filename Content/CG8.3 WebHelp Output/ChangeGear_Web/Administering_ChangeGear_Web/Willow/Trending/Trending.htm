Overview of Trending

Trending is the grouping or clustering of similar tickets that were created within a specified duration of time. This feature utilizes Serviceaide Willow AI™ to determine trends in ChangeGear. Trends or Events allow IT Staff to quickly view and address an issue affecting multiple users and/or provide a workaround. Trends allow for end users to readily view current events in ChangeGear and associate their issue to an existing trend with ease using 'Plus Me'. As new tickets are opened they will be added to an open trend if Serviceaide Willow AI™ identifies them as being similar to that event.
