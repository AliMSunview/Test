Smart Query Overview

Smart Query is a separate page in ChangeGear that enables users to search for information independent of a ticket.

  • Smart Query is available to all ChangeGear users (Staff and End Users).

  • Users click the Smart Query icon  to open the page.

  • ChangeGear supplies a default configuration for Smart Query. It can also be enabled or disabled.

  • Administrators set up and configure Smart Query using the Willow > Smart Query node inside the ChangeGear Web Administration.

  • Smart Query has a list of data sources associated with it. Each data source is associated with a Willow model (restricted to text models only) and has other properties like threshold, number of rows to show, title and body fields to be used, etc.

  • If a Willow model associated with a source is deleted, ChangeGear deletes the link between the model and the source as well.
