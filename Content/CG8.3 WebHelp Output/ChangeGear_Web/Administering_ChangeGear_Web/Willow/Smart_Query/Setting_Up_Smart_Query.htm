Setting Up Smart Query

To set up Smart Query for users, perform the following steps.

  1. Click Web Administration>Willow > Smart Query. ChangeGear displays the Smart Query screen.

  1. To set up the sources for Smart Query, click Data Sources. ChangeGear opens the slide-out editor.
  2. To add the first source, click + Sources.

Enter a Name for the source.

Select the source's Model type from the dropdown. ChangeGear lists only the Willow types available per your license.

Enter the Number of rows to display on the Smart Query table view UI. Determines how many results that ChangeGear shows to the staff user in the Smart Query screen.

Title Field is the what ChangeGear uses from the source to display as the title on the Smart Query table and list views. Your options for the title varies based on the type of model chosen.

Body is what ChangeGear uses from the source to display as the body on the Smart Query list view. Your options for the Body varies based on the type of source used.

Threshold determines what results ChangeGear classifies as high, and displays a green bar to the right of the result. The default threshold is 60. ChangeGear does not displays any result with a threshold below what you set in this field.

  1. Next, customize the Header HTML. This is the text that appears above the Smart Query search field in the initial page that ChangeGear displays to the user. 

Type the text that you want to appear. Then set the HTML settings, such as font type and size, bold, centered, underlined, color, etc. 

  1. Type the text that you want to appear. Then set the HTML settings, such as font type and size, bold, centered, underlined, background (the default is dark, but you can choose light as well), etc. 
  2. When you have finished setting up the sources and Header text, click the save button.
  3. To enable Smart Query, slide the enable button to the right and save your changes.
