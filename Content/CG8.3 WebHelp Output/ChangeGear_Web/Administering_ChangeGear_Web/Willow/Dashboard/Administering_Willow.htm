Administering Willow

Willow is an automated response system that helps end users resolve issues without having to wait for a response from a staff user. Using existing help requests and knowledge base (KB) articles, Willow can provide potential solutions to incidents before users have to submit a ticket.

You can administer the settings of Willow to enable/disable features.
