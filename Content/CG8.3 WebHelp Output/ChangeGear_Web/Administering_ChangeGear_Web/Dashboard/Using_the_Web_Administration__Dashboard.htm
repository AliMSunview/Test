Using the Web Administration Dashboard

The Administration Dashboard is the home page for the Administration module. You can access it at any time by clicking the Administration link.

The Administration Dashboard shows the most essential ChangeGear server information on a single screen, making it easier to gather information needed for troubleshooting or other administration purposes. In addition to server configuration information, the Dashboard also provides tools for working with the ChangeGear server.

Administration information available in the dashboard:

  • Product Information: ChangeGear edition, build information, and server name

  • Web configuration: key web addresses, plus the status of the Intelligence Pack (if licensed)

  • License information: type, end date, activation key, license count, etc.

Administration functions available in the dashboard:

  • Restart the ChangeGear server or the ChangeGear website. These work the same way as they do in the legacy administration client.

  • Send log files or configuration files to yourself via e-email, so that you can easily view the files without needing disk-level access to the ChangeGear server. To use this function, your ChangeGear server account must have an e-mail address on record. To minimize the chance of messages being blocked due to attachment size, each file is sent in an individual e-mail message.
