Identfying Company Locations

Company Locations identifies your organization's locations by city.

  1. On the Web Administration page, click Platform > Company Info > Location. ChangeGear displays the Locations screen.
  2. To add a new location, click the plus sign in the upper right corner. ChangeGear opens a row in the grid so you can enter the location's Name and Description.

Note: You can add more columns to your grid to add optional location information: Created, Editor Layout, Location ID, Modified, and Owner.

  1. To edit an existing location's information, hover over the line and click the pencil on the right. ChangeGear opens that row in the grid so you can change the information.
  2. To delete a location from the grid hover and click the trashcan icon on the right.
  3. When you have finished adding or editing location information, click the save button or the right.
