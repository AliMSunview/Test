Translating the  Page

You can use the Translate Page option (available on the Customize slideout) to translateall contents of the page.

1.  Go to Administration>Self-Service Portals to display any existing portals that have been created.

2.  Click  next to the SSP you want to translate to display an editing slideout,

3.  Enable the Translate Page option.

4.  Click on the save icon () to store the setting.
