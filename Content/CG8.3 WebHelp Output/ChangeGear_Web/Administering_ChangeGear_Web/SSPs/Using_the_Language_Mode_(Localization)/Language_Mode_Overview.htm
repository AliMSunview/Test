Self Service Portal Language Mode Overview

You can easily localize elements that appear on your Self Service Portal (SSP). Each SSP can contain localized text that is completely separate from other portals.

An SSP can be translated into multiple languages with users automatically directed to their language based on their browser settings. Widget titles, description etc., will be translated. Content displayed in the widgets, like announcements, ticket titles, etc, will not be translated using this feature.

There are two options for localizing these pages:

  1. Language Mode for translating just widget titles and their descriptions.
  2. Translate Page Option for translating content displayed in the widget.

Notes:
-There are 6 languages included; other languages can be added via app config and adding strings.
-While in Language mode, you cannot edit the layout.
-This feature is not available for factory default Self Service Portals.
-ChangeGear utilizes Google Translate for all localization features. Google Translates limits the number of automatic/suggested translations you can do within a 1-hour period. However, manual translation is unlimited.
-Repeat the processes shown below for each language you want to make available.
-When accessed, SSPs automatically grabs current locale of the browser.
