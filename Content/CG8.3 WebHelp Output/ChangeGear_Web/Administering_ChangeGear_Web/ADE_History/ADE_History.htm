ADE History

ADE History grid displays the operations that are run on the ADE tool. The main operations include “Discover” and “Import” and are logged on the server. 

Note: In instances where the ADE tool is installed and run-on isolated networks or domains and is not able to reach the service, these history logs will not be available.

ADE Discovery Notifications

ADE has the ability for users to receive notifications on the status of the Discovery operations. The notification will include the start, end and execution time of the scan and the status.
