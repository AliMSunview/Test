Problem Tabs

The tabs at the bottom of the problem form provide more tools that you need to effectively manage your problems.

Description

Enter any communications and/or notes in the Description tab. The Description field captures the name of the person adding to the fields, as well as the time and date of the new entry. If your ChangeGear administrator has enabled the Conversations feature, you can also send correspondence to users via e-mail directly from the form. For more information, see Using Conversations. Once the ticket is saved the entries are listed chronologically and are read only.

Note: The Description field can be seen by End Users.

Enter the resolution to the problem in the Resolution field.

Related Items

Search for related items - such as problems, other incidents, and KB articles - and link to them. You can also unlink items that do not apply. See Linking to Related Items for more details.

Attachments

Add or view attachments that help to resolve the problem. See Attaching Items for more information.

History

ChangeGear keeps a history of the changes made to a ticket. You can add history items, such as when you last tried to contact a requester to the incident report. See Managing History for more information.
