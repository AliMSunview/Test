Problem Workflow Status Bar

ChangeGear uses a graphical representation to show the progression of the problem through the problem management process. This workflow status bar tracks the entire process from the submission of a problem report (New) throughout the life cycle until the problem is successfully closed (Closed).

Click on the options below to learn more about the work flow status bar:

  • New

  • Processing

  • Pending Close

  • Closed
