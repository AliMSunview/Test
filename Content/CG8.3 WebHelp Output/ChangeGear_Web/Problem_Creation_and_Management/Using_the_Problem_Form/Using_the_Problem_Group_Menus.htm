Using the Problem Group Menus

Views

The Views group menu lets you select unique views for the Incident module. The My Views menu allows you to view tickets assigned to you or that you have created. The Global Views menu allows you to view all tickets in the system. The views in Global Views are especially helpful to assess tickets that do not have a specific assignee.

See Viewing your Tickets for more information.

Other

The Other group menu links to important areas not immediately accessible from the main ChangeGear Web interface.
