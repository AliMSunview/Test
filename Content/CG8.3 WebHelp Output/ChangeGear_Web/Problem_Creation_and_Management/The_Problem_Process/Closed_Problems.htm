Closed Problems

A closed problem report requires no further consideration and its history is complete. However, you can reopen and reassign a Closed problem. You can type details in the comment dialog and reassign the problem to another change owner.

To reopen a closed problem report:

  1. Click the problem report to reopen.

  2. In the toolbar, click Reopen.

  3. In the comment dialog, type an explanation for reopening the problem.

  4. In the Assigned To field, click the user icon and select the user from the drop-down menu.

  5. Click Submit.
