Attaching Items

ChangeGear lets you attach documents such as test plans, screen shots, floor plans, directly to tickets.

Note: The default size limit for file attachments is 100 MB.

To attach items to a ticket:

  1. Open a ticket from within the ticket module, (e.g., Incident module).

  2. Click the Attachments tab.

  3. Click Add. The Attachment Upload dialog displays.

  4. In the Attachment Upload dialog, click Browse.

  5. Select a Type from a list of the types of attachments.

  6. Enter a descriptive name of the attachment. Entering a descriptive name allows users to quickly know the purpose of the attachment, especially if the file name is too long to be easily viewed in the file name field.

  7. Click OK.
