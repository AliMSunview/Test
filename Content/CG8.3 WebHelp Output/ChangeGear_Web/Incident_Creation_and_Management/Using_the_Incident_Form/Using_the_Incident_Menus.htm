Using the Incident Menus

Views

The Views menu lets you select unique views for the Incident module.

  • The My Views menu allows you to view tickets assigned to you or that you have created.

  • The My Team Menu allows you to view tickets assigned to your team.

  • The Global Views menu allows you to view all tickets in the system. The views in Global Views are especially helpful to assess incidents that do not have a specific assignee.

See Viewing your Tickets for more information about views.

Other

The Other menu links to important areas not immediately accessible from the primary ChangeGear navigation menu.
