Using the Incident Toolbar

The Incident Toolbar lets you complete tasks throughout the incident management life cycle. The toolbar provides command buttons in the form of icons.

Note: Some of these icons are only available when you are performing unique processes in the life cycle.

Icon

Description

Saves the Incident.

Cancels changes made to an Incident and closes the form.

Creates a new Incident. When used within the Incident form, it creates a new Incident in the same window.

A drop-down option to create a ticket from a template is available if you have created a template. See Creating and Using Templates for more information.

Creates a duplicate Incident. When used within the Incident form, it creates a duplicate Incident in the same window.

Prints the Incident.

E-mails the Incident.

Displays a link to the Incident.

Opens a dialog that allows you to search for other tickets or articles by module. You can also add an ticket or article from this dialog as a related item.

Submits the ticket to ChangeGear and begins the Incident life cycle.

Accepts the Incident and sets the status to In-Progress.

Promotes the Incident to a Problem or RFC.

Resolves the Incident.

Closes the Incident.

Reopens the Incident.

Sends an e-mail to the requester for more information.

Either creates a template from the current ticket by using "Save As" or overwrites the current ticket with the selected template.
