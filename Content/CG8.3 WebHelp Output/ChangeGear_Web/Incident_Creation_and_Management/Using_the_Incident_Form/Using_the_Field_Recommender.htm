Using the Field Recommender

As you fill out a new ticket form, ChangeGear can intelligently determine the type of ticket that you are creating. The Field Recommender is an automatic process that suggests values for your ticket using the values entered in similar tickets. If there are a lot of tickets similar to yours–for instance, if this is a frequently occurring problem–then the Field Recommender can accurately anticipate what the ticket type is, what the severity is, and who the assignee will be.

The Field Recommender uses a self-learning process to automatically improve its ability to provide relevant suggestions over time. As more tickets are created, the Field Recommender uses the data from those tickets to better understand how tickets are categorized, prioritized, and assigned. This in turn increases its confidence that the values it suggests are the most appropriate values for that ticket. You can override the values suggested by the Field Recommender at any time.

The Field Recommender feature is available for Incident Requests that are initiated by end-users.  When tickets are submitted by end-users, field recommendations determined by the Willow Text Similarity and Classification models are automatically applied to the smart recommend enabled fields on the ticket that does not have user input. This  provides for actionable requests to be created which facilitates  faster resolution of the tickets by the staff users.

Note: This feature may not be available depending on your ChangeGear installation.

To use the Field Recommender:

  1. Open a new ticket.

  2. Enter a value in the Summary field. As you type, the Field Recommender provides a drop-down list of summaries from similar tickets.

  3. Select a ticket from the drop-down to auto-populate the current ticket's fields with the same fields from the recommended ticket.

  4. And new in ChangeGear 8, when you select an Incident Type, ChangeGear also recommends values for the Assigned To, Impact, Urgency, and Priority fields.

Confidence Levels

The Field Recommender indicates the relevance of each recommendation using a confidence level. The confidence level indicates how certain the Field Recommender is that this is the appropriate value for this field based on similar tickets. Confidence is measured on a scale from 0-100% and is represented by an indicator at the top-right corner of the field. The Field Recommender uses a color scale to indicate confidence levels:

  • Green indicates a high confidence.

  • Yellow indicates a moderate confidence.

  • Red indicates a low confidence.

  • Gray indicates a low confidence, or that no match was found.

The Field Recommender not only generates a confidence level for the suggested solution, but it generates a confidence level for other values in the same field. It then compares each confidence level to determine the best fit. Values that have a significantly higher confidence level than other values are more likely to be recommended, even if the confidence level isn't necessarily high on its own.

To see the possible suggestions provided by the Field Recommender, click on the indicator.

You can override an auto-populated field by clicking on an alternate value in the indicator, or by manually changing the value of the field. If the new value differs from the Field Recommender's suggestion, ChangeGear displays an exclamation point over the indicator.
