Predicting Trends

The Reports module includes several reports for predicting Incident ticket trends. These reports can be found in the Incident Metrics category.

Note: This feature may not be available depending on your ChangeGear installation.

  • Incidents Per Day predicts the number of new tickets that will be opened per day.

  • Incidents Assigned Per Day Per Team predicts the number of tickets that will be assigned to different teams per day.

  • Incidents Assigned Per Day Per User predicts the number of tickets that will be assigned to different staff users per day.

  • Incidents Closed Per Day predicts the number of tickets that will be closed per day.

  • Incidents Closed Per Day Per Team predicts the number of tickets that will be closed per day per team.

  • Incidents Closed Per Day Per User predicts the number of tickets that will be closed per day per user.

  • Incidents Created Per Day Per Type predicts the number of new tickets that will be opened per day, grouped by ticket type, with each ticket type displayed on a different line.

Each report displays the number of items leading up to today's date, followed by the number of items anticipated for future dates.
