Viewing Suggestions

ChangeGear can use the contents of existing tickets to expedite the creation, management, and closing of new tickets. This ability allows features such as the Field Recommender and the Suggestion Center to provide recommendations based on data that has already been entered into other tickets. While these features are accessed through the Incident ticket form, you can also view suggestions through the Incident module grid view.

Note: This feature may not be available depending on your ChangeGear installation.

The Incident module ticket grid provides two columns for viewing suggestions:

  • Suggestion displays ChangeGear's confidence in providing suggestions for the ticket. When comparing a ticket to other tickets, ChangeGear rates each ticket based on its similarity to the current ticket. Tickets that are more similar are rated first, whereas tickets that are dissimilar are rated last. The Suggestion column represents ChangeGear's confidence that the top-rated ticket is the most relevant to this ticket. For more information on confidence levels and suggestions, see Confidence Levels under Using the Field Recommender, and Using the Suggestion Center.

  • Est Hours displays the number of hours and minutes expected to resolve the ticket. This field is calculated based on the amount of time taken to close similar tickets, and is updated each time the ticket is saved.

    Important: This field represents an estimated time, not a guaranteed time. This value does not factor in or replace any SLAs that are currently active.

These columns can be added to any view. They can also be found in the global Smart View view. For more information on adding columns to a view, please see Using the Grid > Adding and Removing Columns.
