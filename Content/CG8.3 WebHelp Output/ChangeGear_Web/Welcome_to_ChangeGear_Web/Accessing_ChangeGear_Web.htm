Accessing ChangeGear Web

You can access ChangeGear Web using a web browser.

To access ChangeGear Web:

  1. Launch your web browser.

  2. In the address bar, type: http://<ChangeGear server>/CGWeb where <ChangeGear server> is the name of the server where ChangeGear is installed. After entering this URL in your browser you should see the ChangeGear Web home page appear.

    Tip: If you have just installed ChangeGear Web and are having difficulty accessing it, try using the ChangeGear Web shortcut that was placed in the Start menu under Programs > ChangeGear.

  3. Observe that ChangeGear Web can automatically log you in using the credentials you used to log in to the Windows network; even though you do not type a user ID and password, this use of Windows integrated authentication provides a secure environment.

  4. Be aware that, depending on your browser settings, you still may be prompted to enter your Window's network credentials. If you are prompted, enter the same domain, user Id, and password that you enter when logging into your computer.

To display the ChangeGear main menu:

After logging in to ChangeGear Web, click on the icon in the left most corner of the Title bar to view the ChangeGear Primary Navigation Menu.

All modules available for your role display in the menu from which you can navigate to the appropriate module.
