Overview of ChangeGear Web

ChangeGear Web provides a versatile Self-Service Portal that can be accessed from a web browser. ChangeGear Web focuses on the needs of both the Staff and End User and allows users in either of these roles to act remotely without requiring any software installed other than a browser.

ChangeGear Web provides a list of service announcements acting as a bulletin board informing you about IT service changes and outages.

Note: Some actions, items, or modules may not be available due to license restrictions.
