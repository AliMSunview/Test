Changing Your ChangeGear Password

You can change the password you use to login to ChangeGear from your User Profile page.

Note: You can only change your password if you are a ChangeGear authenticated user (i.e. you were given a new user name and password to access ChangeGear). If you are logging into ChangeGear with your Windows Integrated Authentication account then you will not be able to change your password within ChangeGear. Please contact your IT department or ChangeGear Administrator if you have questions about your account.

To change your password:

  1. Open your user profile page and click on the Change Password button at the bottom of the page.

  2. The Change Password dialogChange Password dialog will appear.

  3. Enter your current password.

  4. Enter your new password, and then enter it again to confirm it.

  5. Click Submit.

  6. Your password has now been changed and must be used the next time you login.
