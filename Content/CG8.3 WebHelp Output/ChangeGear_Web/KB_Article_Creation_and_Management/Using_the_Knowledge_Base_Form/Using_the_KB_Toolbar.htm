Using the KB Toolbar

The Knowledge Base Toolbar lets you perform common operations by clicking command buttons in the form of icons.

Icon

Description

Saves the article.

Creates a new article. When used within the KB form, it creates a new article in the same window.

A drop-down option to create an article from a template is available if you have created a template. See Creating and Using Templates for more information.

Creates a duplicate article. When used within the KB form, it creates a duplicate article in the same window.

Deletes the article.

Opens a dialog that allows you to search for other articles. You can also add an article from this dialog as a related item.

Prints the article.

E-mails the article.

Displays a link to the article.

Sends a review request to the selected reviewer.

Publishes the article.

Revises the article.

Pulls back the article.

Retires the article.

Either creates a template from the current article by using "Save As" or overwrites the current article with the selected template.
