Using the KB Group Menus

Views

The Views menu lets you select unique views for the Knowledge Base module. You can use the menu to view all articles, recently updated articles, and the most popular articles.

See Viewing your Tickets for more information.

Other

The Other menu links to important areas not immediately accessible from the main ChangeGear Web interface, such as the Announcement Portal and the Self-Service Portal.
