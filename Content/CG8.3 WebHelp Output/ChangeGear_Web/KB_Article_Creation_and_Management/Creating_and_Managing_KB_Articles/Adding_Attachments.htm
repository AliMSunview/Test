Adding Attachments

You can add attachments to KB articles through the Attachments tab. ChangeGear can attach any file type to an article including documents, images, and screen shots. With the exception of embedded images, users can view a list of attachments at the bottom of the article.

To add an attachment to a KB article:

  1. Create a new KB article

    - OR -

    Open an existing KB article.

  2. Click on the Attachments tab.

  3. Click Add to open the ChangeGear Attachment Upload dialog.

  4. Click Choose File to select a file to upload.

  5. Select a file type.

    Note: The Embedded Image type is for images that are inserted into the body of the article. End users will not be able to see embedded files in the attachments list.

  6. Enter a descriptive name for the attachment. Entering a descriptive name allows users to quickly know the purpose of the attachment.

  7. Click OK.

  8. Click the button.

You can verify that your file was successfully attached in the Attachments tab.

Embedding Images

The only exception to using the Attachments tab is when embedding images. Embedded images are attached by inserting the image directly into the body of the article. You can view the embedded image in the attachments tab, but end users will not see the image in the list of attachments.

To embed an image into a KB article:

  1. Create a new KB article

    - OR -

    Open an existing KB article.

  2. Make sure the Body tab is selected.

  3. Click on the location in the article where you would like to embed the image.

  4. Click the button to open the Insert Image dialog.

  5. Click the Browse... button to upload an image from your computer

    - OR -

    Link to an image on the Web by selecting the From the web (URL) radio button and entering the address of the image.

  6. Optionally, click the More Options checkbox to change the size, position, and description of the image.

  7. Click Insert to add the image to the article.

Tip: Users can still download embedded images when viewing the article by right-clicking the image and selecting Save As.
