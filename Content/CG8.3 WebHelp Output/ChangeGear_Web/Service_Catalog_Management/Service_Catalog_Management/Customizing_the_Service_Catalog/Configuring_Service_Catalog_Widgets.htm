Configuring Service Catalog Widgets

The Service Catalog features several configurable widget types. Each widget type has configuration options. 

Service Navigator Widget

The Service Navigator widget is the backbone of the Service Catalog. All published services will show in this widget, and users will search, browse and order services from here. Only one copy of the Service Navigator widget can exist on the Service Catalog. 

Top Requests Widget

This widget will show the most popular services requested over a configurable period of time. The number of days to count and the number of service requests to show are configurable. Only service items that were submitted as actual Service Request tickets via the Service Navigator are counted.   

HTML Widget

This widget will display custom HTML code. Enter the code and save the changes to preview the output. Select the "Display as banner" option to make the widget display with no borders, so that it can be used as a banner image.   

Links Widget

The Links widget contains a list of HTML links. 

Tickets Widget

The Tickets widget shows a list of tickets from a specified module. The list can combine tickets from more than one view.  When viewing the widget on the Service Catalog, the user can click a ticket on the widget to open the ticket.   

Note:  The two fields named "Display Field 1" and "Display Field 1" must be typed manually, and must match existing ticket text field names.

 

Views Widget

The Views widget shows a list of views from a specified module. For each view, the widget can be configured to show the count of tickets in that view.  When viewing the widget on the Service Catalog, the user can click a view name to open a new tab that shows the tickets in the view.
