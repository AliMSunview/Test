Asset Overview

Asset module provides you with the tools you need to monitor and control your organization’s assets throughout their life cycle. An asset’s life cycle may include requesting, ordering, invoicing, shipping, receiving, paying, imaging, deployment, lease vs. purchase, warranty, depreciation, and retirement/end-of-life of the asset. Asset Management optimizes cost while minimizing risk by ensuring all of the following:

  • Continual tracking of each asset and its attributes such as value (including depreciation), warranty, contracts, etc. You always know where an asset is located, who is using it, how it’s being used, how it’s configured, what it costs, as well as its delivered value, etc.
  • Efficient and accurate purchasing and inventory practices. For example, if a user is requesting a new laptop, IT needs to have an accurate record of laptops in inventory; do they have a laptop available in existing inventory or do they need to purchase a new laptop?
  • Comprehensive reporting that allows you to easily access all necessary information for an asset.

An asset is always associated with an Asset Type. Asset Type drives the Asset Model which in turn allows for depreciation Type etc. to be set on the asset. ChangeGear includes factory Asset Types, and new Asset Types can be added through the Web Administration.

Note: Asset is a licensed module.
