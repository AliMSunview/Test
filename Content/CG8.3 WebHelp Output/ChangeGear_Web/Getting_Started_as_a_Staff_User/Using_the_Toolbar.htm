Using the Toolbar

The tool bar lets you perform a variety of functions in the grid view.

Icon

Description

Saves the Incident.

Cancels changes made to an Incident and closes the Incident form.

Opens a new ticket. A drop-down option creates a ticket from a template when available. See Creating and Using Templates for more information.

Creates a duplicate Incident. When used within the Incident form, it creates a duplicate Incident in the same window.

Prints the selected ticket.

E-mails the selected ticket to whomever you designate.

Displays a link to the ticket.

Deletes a ticket. You can delete multiple tickets at a time by selecting a contiguous block. Selecting a starting point and click on an end point selection while holding down CTRL+Shift. Select non-contiguous tickets by CTRL+click on each selection.

Edits an existing ticket.

Refreshes the grid.

Opens a dialog that allows you to search for other tickets or knowledge base articles by module. You can also add an ticket or article from this dialog as a related item.

The Manage View menu also allows you to save or modify customized views.

Click on Manage View to open a menu that contains controls for the view. You can save modification you have made to the view, modify existing views, delete custom views, and show/hide columns. You can also export the grid into excel, PDF, RTF, or CSV formats.

The View List menu allows you to select unique ways to view the information in the grid, including pre-filtering or sorting the items to only those items you wish to view. You can also view your custom views from this list.

Quick search is located next to the search text box and allows you to search in Commonly Used Fields of your current view. You can use the down arrow to refine your search fields in the Quick Search.

The page navigation buttons allow you to move to the first page, the previous page, the next page, and the last page of the grid.
