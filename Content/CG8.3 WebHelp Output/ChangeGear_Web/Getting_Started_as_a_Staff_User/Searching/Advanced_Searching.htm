Advanced Searching

In addition to searching tickets, certain areas of ChangeGear allow you to search Knowledge Base articles, announcements, managed resources, and service items using a simple but powerful syntax. This feature can be used in the Self-Service Portal, the SmartHub, and the Workspace. Intelligent Search automatically indexes string fields such as ticket summaries, Knowledge Base article titles, and ticket history records. This also includes custom string fields.

Note: You may not have access to this feature depending on your ChangeGear configuration.

To perform a search:

  1. Open the ChangeGear primary navigation menu and select a module

    - OR -

    Navigate to the Self-Service Portal or Workspace.

  2. Enter your search terms using the guidelines defined below.

  3. Optionally, enter any operators that you wish to use to specify your search criteria. See below for a table of operators and their effect on your search results.

  4. Press the Enter key or click the search icon to display a list of search results.

  5. Click on an item to view it.

Search Guidelines

Search Guideline

Example

Single Word

Patch searches for the word Patch regardless of letter case.

Stop words

Patch an management returns any matches containing patch or management.

Inflections

Patch returns matches for patches and patching.

Phrase

Patch management returns matches for patch manage and patch managing.

Note: The search replaces special characters that are not operators with an empty space. For example, if you enter "bounce-test:server" as the search string, Intelligent Search replaces the dash and the colon with spaces, resulting in any items containing the words bounce OR test OR server.

Search Operators

Search operators allow you to enhance the flexibility of your searches. Intelligent Search supports the following operators:

Operator

Description

Example

AND (&&)

Returns results that include both search terms.

Printer && offline returns Printer is offline, but not Computer is offline.

OR (||)

Returns results that include at least one of the search terms.

Printer || offline returns both Printer not working and Computer is offline.

Prefix (*)

Returns results that begin with the search term.

Down* returns both downtime and download.

Fuzzy (~)

Returns results that are similar to–but do not exactly match–the search term.

Stnd~ returns stand, sand, etc.

Exclude Word (-)

Returns results that do not contain the term.

Telephone -Internet returns results containing telephone, but not Internet.

Files Searchable by Intelligent Search

Intelligent Search parses the contents of attachments with the following file formats:

  • Word documents (.doc, .docx)

  • PowerPoint presentations (.ppt, .pptx)

  • Excel spreadsheets (.xls, .xlsx)

  • Plain text files (.txt)

  • Rich-text files (.rtf)

  • Portable Document Format files (.pdf)
