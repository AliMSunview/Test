Setting a Default Change Mode

Staff Users can bypass the Change Mode selection prompt by specifying a default Change Mode in your ChangeGear preferences. For more information, see the Setting ChangeGear Preferences topic under Welcome to ChangeGear Web. ChangeGear administrators can also set a global default Change Mode, although your preferences will override this setting. End users will always use the global default when submitting new change requests.

The global default will always apply when creating a change request from the Self-Service Portal, Service Catalog, or User Center.
