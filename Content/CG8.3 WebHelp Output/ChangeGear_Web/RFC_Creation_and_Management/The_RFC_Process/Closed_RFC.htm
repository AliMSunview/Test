Closed RFC

A closed RFC requires no further consideration and its history is complete. You can select to reopen a Closed request from the toolbar. You can type comment details in the comment dialog and re-assign the RFC to another change owner.

To reopen a closed RFC:

  1. Click the RFC to open.

  2. In the toolbar, click Reopen.

  3. In the dialog, type an explanation for reopening the RFC.

  4. In the Assigned To field, click the user icon and select the user from the drop-down menu.

  5. Click Submit.
