Assigned RFCs

Once a change request is approved, ChangeGear automatically changes the status of the request from Pending Approval to Assigned. You can use the RFC form to review your deadlines and indicate that you have started the implementation. After your change manager assigns you to a change request, you might have a deadline for beginning implementation. You can use the RFC form to review your deadlines and indicate that you have started the implementation.

To start the implementation of an RFC:

  1. Click the RFC to open.

  2. Review the RFC data fields, click Accept.

  3. In the Comments field of the Web Page Dialog, type any comments you have.

  4. Click OK to save your changes.

  5. Observe the RFC is not in the In-Progress state.

Creating New Tasks

When you accept a change request, ChangeGear systematically notifies the change management staff that you have begun its implementation. This notification lets the change manager know that you are starting the tasks necessary to complete the request and moves the state to In-Progress. However, the Implementation tab provides no record of your actions until you create a task. With each task record, you and other assignees can provide a detailed account of your implementation activities. For example, you can record a description of the tasks, identify the person that performed it, and account for its cost in time and effort. You can also update the status of each task as you take it from start to completion.

To create a new task:

  1. Click the RFC to open.

  2. At the RFC Tabs, click the Implementation tab.

  3. In the Implementation tab toolbar, click New.

  4. In the Task, select the Implementer from the list -or- click to select dynamic role assignment for Implementer and/or other types of Implementers.

  1. In the Description field, type a brief description of the new task.

  2. In the Type list, select the appropriate implementation task type.

  3. In the Status list, select New.

  4. In the Start fields, select the start date and times of the implementation task.

  5. Enter the Duration of the task.

  6. In the Complete fields, select the start date and times of the implementation task.

  7. Enter the Effort of the task.

  8. In the Notes box, type any explanations or task requirements.

  9. Click Save.
