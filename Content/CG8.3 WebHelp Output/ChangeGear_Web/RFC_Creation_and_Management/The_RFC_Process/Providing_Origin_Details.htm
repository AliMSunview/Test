Providing Origin Details

ChangeGear lets you track external IDs linked to RFC whether they're related to your customers, partners, or internal organizations such as your Help Desk.

To provide origin details:

  1. In the RFC form, click the button adjacent to the Origin field. The Origin Editor appears.

  2. In the Origin list, select the origin of the change request.

  3. In the Origin ID, type the source ID (for example, a Help Desk ticket ID number).

  4. In the Origin Notes box, type any details or notes to link the change request to the origin.

  5. Click OK to add the origin details.
