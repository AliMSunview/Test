Implementation Tab

Implementation is the assignment, planning, and execution of tasks to satisfy the requirements of an approved change request. The RFC form lets you account for critical items in the change implementation process such as the actual start date, the actual duration of the change request, the total resource costs, and any status changes to sub-tasks needed to complete the change request. The change owner uses this tab to plan individual tasks and make further IT staff assignments. Every completed tasks record updates the Implementation tab.

You can change the Status of the task by selecting status from the drop-down menu.

To create a new task:

  1. Open an RFC.

  2. Click the Implementation tab.

  3. In the Implementation tab toolbar, click New.

  4. In the Tasks window, click the Implementer drop-down.

  5. Select your name from the list

    -OR-

    If you are creating the task for another person, select the name of the person for whom you are creating the task.

  6. Complete the rest of the Tasks window and click Save.
