Attachments Tab

The Attachments tab lets you add, remove, and review files supporting anything you deem necessary for the change request. You can adjust the column widths and sort the columns.

To add an attachment using the Attachments tab:

  1. Open an RFC.

  2. Click the Attachments tab.

  3. In the Attachments toolbar, click Add.

  4. In the Browse Attachments dialog box, if you want to attach a document(s), click the Attach radio button and click Browse to locate the file

    -OR-

    If you want to link your attachment instead of copying it to your RFC, click Link to.

    Note: When you link an attachment to the RFC, any changes or updates to the source file are automatically updated in your RFC. Or, if you don't link an attachment to the RFC, and just add an attachment instead, all changes to the original source file will have no effect on your attachment. This is an effective way to create an archive of your test plans, etc.

  5. In the Type drop-down, select the reason for the attachment and click OK.

  6. Enter a descriptive name of the attachment. Entering a descriptive name allows users to quickly know the purpose of the attachment, especially if the file name is too long to be easily viewed in the file name field.

  7. You can verify your file is attached in the Attachments tab.

To remove an attachment:

You can remove obsolete, erroneous, or irrelevant files. You can also save HDD space by removing files. The Size column displays the size or HDD space requirement of the attached file.

  1. Open an RFC.

  2. In the RFC form, click the Attachments tab.

  3. In the Attachments list, select that attachment or attachments you want to delete.

  4. Click Remove.

  5. Confirm your removal at the confirmation window.

To view an attachment:

  1. Open an RFC.

  2. In the RFC form, click the Attachments tab.

  3. In the Attachments list, select the attachment you want to delete.

  4. Click View.
