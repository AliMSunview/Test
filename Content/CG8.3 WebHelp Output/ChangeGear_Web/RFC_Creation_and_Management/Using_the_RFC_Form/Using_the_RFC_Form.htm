Using the RFC Form

The ChangeGear Request for Change (RFC) form provides a graphical foundation for documenting and considering a change request. The RFC form lets you update the status of the change request as it progresses through the change management life cycle.

The RFC form is comprised of these parts:

  • RFC Toolbar

  • RFC Workflow Status Bar

  • RFC Data Fields

  • RFC Tabs
