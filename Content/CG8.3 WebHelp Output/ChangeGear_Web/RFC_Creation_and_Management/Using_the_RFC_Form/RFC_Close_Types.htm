RFC Close Types

Close Types are reasons describing the reason a change request was closed. When you close an RFC, you should complete the fields that help determine the total cost of the change request.

  • Duplicate

  • Known Error

  • Resolved - Confirmed

  • Resolved - Unconfirmed

  • Standard Change

  • Unresolved

  • User Training
