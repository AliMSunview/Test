Using the RFC Toolbar

The RFC toolbar lets you complete tasks throughout a change management life cycle. The toolbar provides command buttons in the form of icons.

Note: Some of these icons are only available when you are performing unique processes in the life cycle.

Icon

Description

Saves changes to the RFC or saves the RFC as a draft.

Cancels changes made to an RFC and closes the RFC form.

Creates a new RFC. When used within the RFC form, it creates a new RFC in the same window.

A drop-down option to create an RFC from a template is available if you have created a template. See Creating and Using Templates for more information.

Creates a duplicate RFC. When used within the RFC form, it creates a duplicate RFC in the same window.

Prints the RFC.

E-mails the RFC.

Displays a link to the RFC.

Opens a dialog that allows you to search for other tickets or articles by module. You can also add an ticket or article from this dialog as a related item.

Lets you escalate the change request and change the status to Escalated.

Lets you change the status to Pending Close.

Lets you resubmit the change request and change the status to New.

Lets you close the change request and change the status to Closed.

Lets you defer the status and assign to On-Hold.

Lets you change the status to Need More Info.

Lets you change the status to Requester Review and assign the request.

Lets you accept the change request and move the status to In-Progress.

Lets you submit the change request for approval and change the status to Pending Approval.

Lets you approve the change request and change the status to Assigned.

Either creates a template from the current ticket by using "Save As" or overwrites the current ticket with the selected template.
