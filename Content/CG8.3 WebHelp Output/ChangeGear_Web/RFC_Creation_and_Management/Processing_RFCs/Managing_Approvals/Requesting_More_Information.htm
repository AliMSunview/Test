Requesting More Information

If you review a ticket and determine it lacks sufficient detail for approval, you can request more information from the requester. When you request more information, ChangeGear automatically assigns the request to the requester. At the same time, a predefined notification rule sends a template-based e-mail to the requester which notifies the requester of the additional information request.

To request more information for the ticket:

  1. Open the appropriate ticket.

  2. Click More Info.

  3. In the Comment box, type the complete details of your information request.

  4. Click OK to save.
