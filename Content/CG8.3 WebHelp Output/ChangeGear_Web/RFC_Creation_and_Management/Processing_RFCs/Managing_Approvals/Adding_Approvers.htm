Adding Approvers

If your default role or role within a team has the appropriate privileges, ChangeGear lets you manually add an Approver to a new or existing ticket. Approval notifications are sent in the listed order of the approvers. You can change the order using the move up arrow or move down arrow.

To add approvers to the ticket:

  1. Navigate to the ticket module from the primary ChangeGear Navigation menu.

  2. Open the ticket you want to edit.

  3. Click the Approval tab.

  4. On the Approval tab, click Add. The Select User dialog box displays.

  5. Type the approver’s name in the Type Name box or select it from the list.

  6. Click OK.
