Marking RFCs for Review

If an RFC needs further review before proceeding through the change life cycle, you can mark the RFC for review. Marking an item for review simply sets a flag on the item, making it easier to find and sort using the grid. Once the item has been reviewed, simply remove the flag to return the ticket to normal.

You can view the number of items pending review through widgets on the Dashboard and Workspace. For more information, see Customizing the Dashboard and Customizing the Workspace.

To mark an RFC for review:

  1. Navigate to the Change module from the ChangeGear navigation menu.

  2. Right-click the ticket that you wish to edit.

  3. Click Mark for Review.

To unmark the RFC, simply right-click it again and click Unmark.
