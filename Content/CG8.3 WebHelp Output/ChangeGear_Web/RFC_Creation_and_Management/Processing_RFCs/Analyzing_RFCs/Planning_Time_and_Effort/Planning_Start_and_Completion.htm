Planning Start and Completion

Scheduling the planned start and completion of your change request prevents schedule conflicts with other change requests. You should at least use your RFC schedule to set the expected start and completion dates. If your RFC schedule is complex, you may also want to set the expected start and completion hours.

You can use the calendar to pick Planned Start and Planned Complete dates for the Change ticket that you have open. 

  • You must pick valid dates for ChangeGear to add the time and date you select to the ticket. Valid dates are within a Change Window and outside of a Blackout Period.
  • If you pick invalid dates, such as those that land outside a Change Window or inside a Blackout Period, you must provide a reason for the override to continue. Note: To set invalid dates, your role must have the Override Planned Dates privilege enabled.

  • If you do not have the Override Planned Dates privilege, you cannot pick invalid dates. See your Administrator if you have any questions.

The Change Windows and Blackout Periods shown on this calendar may be different, based on whether those Change Windows or Blackout Periods are global or have criteria.

To plan the expected start and completion of an RFC:

  1. Open any Change Ticket. On the tabs row, click on the Analysis tab to see the Planned Start and Planned Complete dates.

Please go to Generating and Downloading a .ics File for more information about the Add to Calendar feature.

  1. Click on the calendar icon in one of these fields. This opens the calendar at the bottom of the ticket form, and scrolls the form down to show the calendar in full.

  1. The calendar displays the calendar with Change Windows, Blackout Periods, and other Scheduled Changes included.The calendar indicates the first day of each month by displaying the number in orange, and the current day in bold orange.
  2. The calendar shows any day containing a Change Window with a white background and an orange wrench icon  in the upper right corner.
  3. The calendar shows any day containing a Blackout Period with a gray background and a red international no icon  in the upper right corner

  4. To view the times of either a Change Window or a Blackout Period,  hover over the icons to see the start and stop times of each period.

You can also see details of all Change Windows and Blackout Periods on a given day by double-clicking that day's icons. ChangeGear then shows the window/period start and end times, the Name, the Description, and if it is global.

  1. Pick an available day and time on the calendar for the Planned Start or Planned Complete dates. Double click on a day that is not marked as having a Blackout Period to bring up the appointment creation form. This form has the Summary of the current ticket, and date selectors for the Start Date and End Date. Use the date picker to pick any date and time for each of the fields. Note: The Start Date must always be before the End Date. Then click Done; ChangGear puts the dates you selected in the Planned Start and Planned Complete fields.

  1. Save the ticket.
