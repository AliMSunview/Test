Checking Dependencies

Within an IT Department, the set of Business Services offered depends on a complex web of interrelated computers, software, network devices and infrastructure. When you set out to change any part of this interrelated system, it is important to consider these interrelationships, (i.e., dependencies). ChangeGear allows you to map these resource dependencies and view them when assessing the impact of a change.

To View Resource Dependencies:

  1. In the ticket, open the Impacted Resources by clicking the icon or by opening the drop-down menu.

  2. Select the resource or business service for which you would like to check the dependencies.

  3. Click Properties or the button. The resource dialog displays.

  4. Click the Relationships tab.

  5. Expand or contract the relationship nodes to check the dependencies.
