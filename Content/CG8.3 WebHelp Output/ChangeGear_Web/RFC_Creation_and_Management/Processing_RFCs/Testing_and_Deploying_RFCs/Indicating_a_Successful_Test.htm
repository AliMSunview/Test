Indicating a Successful Test

If the change request requires testing, you can’t close it until you indicate that you’ve completed the required tests. ChangeGear provides a check box for you show that you have completed the required testing for the change request. Before you select the testing check boxes, however, you should also document any related testing tasks in the Implementation tab.

To indicate your test was successful:

  1. On the RFC form, click the Release tab.

  2. Determine whether you’re indicating the success of QA or Deployment tests.

    • If the successful test occurred before implementation, select the All tests were successful check box in the QA group.

    • If the successful test occurred after implementation, select the All tests were successful check box in the Deployment group.

  3. Save your changes to the RFC form.
