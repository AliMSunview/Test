Attaching Test-Related Documents

If you’re handling a significant or major RFC, your test plan may also contain a significant amount of detail. For example, your test plan may include floor plans, load tests, or other detailed documentation. ChangeGear lets you attach documents directly to your RFC. Attaching your test documents to your RFCs prevents you from having to look for them later. Your test documentation is located in the Release tab of the RFC.

To attach test-related documents:

  1. On the RFC form, click the Release tab.

  2. Determine the type of test-related documentation you want to attach to the RFC form.

    • If you want to attach a QA test case document, click the Test Cases hyperlink in the Quality Assurance group.

    • If you want to attach a QA test results document, click the Test Results document in the QA group.

  3. In the Add Attachment dialog, click Browse to locate your attachment -or- enter a test case link URL.

  4. Select the file that you want to add and click Open.

  5. In the Type list, select the type of file and click OK.

  6. Click OK.

Note: If you select a type but do not select a file to attach, you see a flashing red exclamation icon to remind you to locate and attach your file.
