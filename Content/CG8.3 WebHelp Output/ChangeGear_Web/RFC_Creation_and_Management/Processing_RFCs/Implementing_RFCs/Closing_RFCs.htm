Closing RFCs

After you determine that an RFC is implemented, you are ready to close it.

The default view of the Change Management module does not display closed RFCs. When you close a change request, however, the Dashboard or Workspace widgets can provide metrics for your change management system.

Tip: ChangeGear lets you create custom views. For example, you could save an existing view as a new view called Closed. You could open the column chooser and add the Status column to the grid. Next, you could filter the Status column to display only closed change requests. Finally, you could save the new view with your column and filter settings. After this, you can select the Closed view from the list in the toolbar to review your closed RFCs.

To close a change request:

  1. In the Actions Tasks group menu, click Close.

  2. In the Close Type list of the RFC Closed dialog box, select the appropriate value.

  3. In the Comment box, type any closing comments.

    Note: You can use the Comment box to record any conditions or reservations for closing the change request.

  4. Click OK.
