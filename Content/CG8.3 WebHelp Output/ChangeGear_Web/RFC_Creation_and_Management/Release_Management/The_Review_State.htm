The Review State

In this stage, the ticket returns to the standard Change Management workflow. It is established if the deployment of the requested change was successful. If the change was successful the process continues.

  • Select the Request Validation option to advance to the Review state.

  • Select the Cancel option to return the ticket to the Assigned state.

Pending Close

After the requester and other stakeholders verify that the change request meets all the success criteria, the Change Manager closes the RFC, noting any necessary comments.
