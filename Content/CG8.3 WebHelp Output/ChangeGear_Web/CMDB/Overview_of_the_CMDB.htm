Overview of the CMDB

With the CMDB, you can track assets, resource types, owners, locations, business services, and users. Generally, this includes hardware, software, infrastructure, policies, procedures, and documentation.

Note: You may not have access to this ChangeGear feature depending on your permissions or ChangeGear edition.

The CMDB is more than just a flat list of resources and properties—you can use this module to track resource dependencies and create a topological model that provides an important knowledge base for your IT staff. After your IT resources are modeled, this knowledge is no longer confined to the collective memory of your staff. It’s electronically available to leverage as part of an integrated service management solution.

Using the CMDB module, you can:

  • Manage CIs

  • View CI details, including configuration details

  • Create and edit CIs using the CI Editor

  • Track owners and users Create tickets related to the CI

  • View items related to the CI

  • Track purchase histories

  • View CI dependencies

  • Scan CIs for configuration changes

  • Add notes to the CI

  • View and manage supporting items, such as SLAs or other contracts

Note: The CI Editor create and edit functions on the web are offered in addition to the CI Editor the ChangeGear Desktop Client.
