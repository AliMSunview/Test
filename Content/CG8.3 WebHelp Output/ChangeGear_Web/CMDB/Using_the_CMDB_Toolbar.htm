Using the CMDB Toolbar

The CMDB toolbar allows you to create, view, and edit CIs; search for related items; and view a selected CI's dependencies.

Icon Description

Creates a new CI. Click on the downward-facing arrow to create a new CI of a specific resource type or to create a ticket with the selected CI set as an impacted resource or business service.

Note: Using the drop down list, you can select an existing CI and create a related ticket that populates the Requester's Item's field automatically. This feature enables you to open tickets and relate CIs directly from the CMDB grid.

Opens the selected CI for viewing or editing.

Creates a new CI by duplicating the currently selected CI.

Opens a dialog that allows you to search for related tickets or articles by module.

Refreshes the CMDB grid.

Opens the Dependency Map.
