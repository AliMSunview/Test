Using the CMDB Group Menus

Views

The Views group menu lets you select unique views for each module. By default, you can select to view all managed items or unmanaged items.

See Viewing your Tickets for more information.

Other

The Other group menu links to important areas not immediately accessible from the main ChangeGear Web interface, such as the Self-Service Portal.
