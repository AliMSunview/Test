Attachments Tab

You can add, view, and remove attachments with information about the resource in the Attachments tab.

To add an attachment:

  1. Click Add. The Attachment Upload dialog appears.

  2. Click Browse.

  3. Select the file you wish to attach to the CI and click Open.

  4. Choose what type of file you are attaching to the CI.

  5. Enter a descriptive name of the attachment. Entering a descriptive name allows users to quickly know the purpose of the attachment, especially if the file name is too long to be easily viewed in the file name field.

  6. Click OK.

The new attachment is listed in the Attachments tab.
