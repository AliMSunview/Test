General Tab

The general tab includes information about the resource that does not fit within other categories.

  • Name: Either the machine name or the DNS name to identify the CI.

  • CI #: This is a unique identifier that cannot be modified.

  • Description: A description of the CI to differentiate between names like HXPNE23 and HXPNE27, (i.e., this is the web server for the intranet).

  • Type: The resource type defines what kind of resource a configuration item is and defines which external applications can be launched from ChangeGear.

  • Version: The version of the CI for record keeping purposes. For example, if the resource is an Apache web server, the version might be 2.0.58.

  • Department: The Department the CI belongs to or that uses it.

  • Status: The life-cycle or deployment state of the CI.

  • Location: The physical location the CI is assigned.

  • Class: The resource class describes the state or classification of the resource, such as managed, unmanaged, tracked asset, or unclassified.
