Configuration Tab

Detailed information about managed CIs are viewable in the Configuration tab. Generally, information is gathered via resource scanning. You can use scanning in conjunction with the Resource Discovery Expert (RDE) to reduce the amount of labor involved with populating and maintaining the CMDB.

Note: Only network devices or machines connected to the network can be scanned.

The Configuration Tab only displays for machines on the network. For example, if you tracked an application, the Configuration Tab would not display. If the Configuration Tab does not display, make sure the Name of the item is the machine name or DNS name.

The Configuration Tab has three sub sections: the Summary tab, the Scan Details tab, and the Scan History tab.

Summary

Information included in the Summary sub-tab varies depending upon what type of CI has been entered or scanned. For example, a router and a desktop machine would have different configuration details recorded.

Scan Details

The Scan Details sub-tab displays the configuration of the CI, which varies between CI types. Use the tree view to navigate to specific items to view the information about the item on the right-hand panel.

Scan History

ChangeGear allows you to view the detailed scan history of your managed CIs. The history shows each time the CI has been scanned and whether or not ChangeGear has detected any changes. You can view the configuration information or compare two different scans for more accurate information on the detected changes. This can help troubleshooting the CI since you have access to the last working configuration versus the new, non-working configuration. See Comparing Scans for more information.

Tracked assets have limited access to the Scan History sub-tab.
