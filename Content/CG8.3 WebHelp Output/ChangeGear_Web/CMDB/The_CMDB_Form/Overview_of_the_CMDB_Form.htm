Overview of the CMDB Form

You can view the details of your CIs and scan network connected devices using the CMDB module in the Web.

Title bar

The title bar contains the CI Name.

Tool bar

Icon Description

Save changes made to the CI.

Cancel any changes made and close the form.

Create a ticket with the CI entered as an impacted resource or business service. Use the down arrow to select which type of ticket to create.

Scan resource details. You may be prompted to enter your credentials.

Note: Only network devices or machines connected to the network can be scanned.
