Creating a Survey

You can create surveys that can be sent for any type of ticket (Ticket Survey) or create surveys without referencing a ticket at all (Stand-alone Survey).

Note: For ChangeGear to automatically send surveys, you must create a BPA.

To create a survey:

  1. Open the Survey module from the ChangeGear primary navigation menu.

  2. Click on New. The Survey Editor displays.

  3. Fill out the form.

    • Name – the name of the survey being created.

    • Title – the text displayed at the top of the Survey Window in the title bar.

    • Introduction – the text for a prompt box that displays prior to the start of the survey.

    • Conclusion – the text for a prompt box that displays upon completion of the survey.

    • Header – an optional item on the survey, which can display additional information about the survey.

    • Footer – displayed at the bottom of the survey.

  4. In the Survey Questions section, click New to create a new question. The Question Editor displays.

  5. In the Question Editor, choose the Question Type from the Type list. Based on the Question Type chosen, the form changes to reflect the context of the type.

    • Satisfaction – This question type is based on a rating , from Very Unsatisfied to Very Satisfied.

    • Open Ended – The survey creator presents a question, and the user can type the answer in a dialog box.

    • True/False – The survey creator presents a question, and the user selects true or false based on the question.

    • Multiple Choice – Multiple choice question type allows the survey creator to give the user options for answering the question.

    • Check List – This question type allows the user to select one or more answers to the question.

  6. Edit the Question.

  7. Edit the Answers.

  8. Select Required if you wish to make answering the question mandatory.

  9. Depending on the Question Type, select Horizontal, when displayed, if you wish the answers to be presented horizontally instead of vertically.

  10. Click OK.

  11. Continue adding questions to the survey by repeating steps 4-10 until you have finished creating the survey.

  12. If you wish to save the survey as a draft, click the Save button in the Survey toolbar at the top of the Survey Editor. You can view the survey in the Draft view or in the All Surveys view of the Survey module.

  13. If you wish to directly publish the survey without review, click the Publish button. You will be able to view the survey in the Published view or in the All Surveys view.
