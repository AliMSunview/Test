Creating and Editing Categories

Categories in the Service Catalog refer to a general way of separating service items into manageable subsections. There are two primary views into the Service Catalog - Business and Technical. Categories can be added to these two view types and can be edited later.

Categories are created and managed in the navigation pane on the Service Catalog module. Categories can only belong to one service type and may be configured to restrict access to all service items within the category. See Accessing Service Items for more information.

To create a category:

  1. Open the Service Catalog module.

  2. In the navigation menu, under Categories, click Add. The Category Editor displays.

  3. Enter the category name in the Name field.

  4. Select the Service Type.

  5. Enter a brief description. This description displays in the Service Catalog Navigator located on the Service Catalog Portal.

  6. If you wish to restrict who can view the category and associated service items, you can select the teams with view capability. See Accessing Service Items for more information on how to restrict access to categories and service items.

    • Choose Selected Teams.

    • Click the  button.

    • Select a team by searching or by using the list.

    • Click Add. The team(s) are added to the bottom grid.

    • Scroll down, and Click OK.

  7. Click the Save button.

To edit a category:

  1. Open the Service Catalog module.

  2. Select a category in the navigation menu.

  3. Click  Edit.

  4. Make any modifications necessary.

  5. Click the Save button.
