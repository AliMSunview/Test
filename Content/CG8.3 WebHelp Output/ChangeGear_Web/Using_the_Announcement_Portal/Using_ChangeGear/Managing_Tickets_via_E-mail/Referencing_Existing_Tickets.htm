Referencing Existing Tickets

ChangeGear can automatically detect whether an e-mail is associated with a ticket without requiring a ticket number. When processing incoming e-mails, ChangeGear performs one of the following actions:

  • If the subject field contains a ticket number surrounded by brackets (e.g., "[IR-0000035]" or "[IR-35]"), then ChangeGear updates the ticket specified.

    Note: This requires certain functionality to be enabled in ChangeGear. Please contact your IT department or ChangeGear administrator for more information.

  • If the subject field does not contain a ticket number, but the e-mail is part of a thread that was used to create a ticket, then ChangeGear updates the ticket created from that thread.

    Important: Be sure to reply to the correct thread. If you reply to a different thread, ChangeGear will either update a different ticket or create a new ticket.

    Warning: In some cases, changing the subject line will prevent ChangeGear from identifying the correct ticket. Avoid changing the subject line unless it contains the ticket number surrounded by brackets.

  • If your e-mail does not contain a ticket number in the subject field and is not part of a thread that created a ticket, then ChangeGear creates a new ticket.
