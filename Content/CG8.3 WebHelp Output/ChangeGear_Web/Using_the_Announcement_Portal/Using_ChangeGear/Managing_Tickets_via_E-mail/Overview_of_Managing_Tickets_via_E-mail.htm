Overview of Managing Tickets via E-mail

ChangeGear's e-mail functionality lets you submit, manage, and process tickets from a regular e-mail client. It provides an e-mail-based alternative to managing ticket life cycles and processes while providing many of the same actions that are available in the ChangeGear Web client. By sending an e-mail to an address specified by your ChangeGear administrator, you can submit requests, respond to incidents, and process requests without needing access to a desktop or laptop computer.

Note: This functionality must be configured before use. Please contact your IT department or ChangeGear administrator for more information.
