Filtering Columns

You can display and hide records in ChangeGear's grid by filtering columns. When you filter a column, the list displays only the records that meet certain conditions that you specify. Each column displays a set of standard values and field-specific values, based on the records currently in the list.

To filter the grid using text filters:

  1. In the column header text boxes enter the text to use for your filter.

  2. Click the filter icon next to the text box and select from the following filters:

    • Begins with - displays only the records that begin with entered text

    • Contains - displays records that contain the entered text

    • Doesn't Contain - displays records that do not contain the entered text

    • Ends with - displays records that end with the entered text

    • Equals - displays only the records that match the entered text

    • Does not equal - displays only the records that do not match the entered text

    • Is less than - displays only the records that display a value less than the one you entered

    • Is less than or equal to - displays only the records that display a value less than or equal to the one entered

    • Is greater than - displays only the records that display a value greater than the one you entered

    • Is greater than or equal to - displays only the records that display a value greater than or equal to the one you entered

To filter the grid using existing field values:

  1. In the column header, click the filter iconnext to the column name.

  2. Check the Select All check box to view all records, or check one or more of the check boxes of the values of the field. For example, if you were filtering records for all tickets requested by George Smith and Bob Johnson, you would check the check boxes next to George Smith and Bob Johnson in the Requester column filter.
