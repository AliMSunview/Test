Sorting the Grid

Sorting items in the grid helps to organize your records in a meaningful way. You can also sort records that have already been sorted without affecting the original list. For example, if you sorted items by Priority to easily view hot work items, you could also sort the items by Due Date to further prioritize your tickets.

You can sort your columns in the following ways:

  • Click a column header. ChangeGear sorts the column in ascending order. A triangle appears on the column header, pointing up.

  • Click the column header a second time. ChangeGear sorts the column in descending order. A triangle appears on the column header, pointing down.

  • Hold the SHIFT key down while clicking in a second column header to sort by a second field.

To remove sort criteria from your column:

  • Hold the CTRL key down while clicking in a sorted column header removes sorting by that column and preserves other sorting conditions.
