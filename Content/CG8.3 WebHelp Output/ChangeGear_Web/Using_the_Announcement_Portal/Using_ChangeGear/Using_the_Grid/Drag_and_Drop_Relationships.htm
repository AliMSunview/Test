Drag and Drop Relationships

You can drag and drop columns in both the column header and in the group panel.

To drag and drop column headers:

  1. Click and hold the left mouse button on the column header you want to move.

  2. While holding the mouse button, drag the column header to the desired location.

  3. Release the mouse button.
