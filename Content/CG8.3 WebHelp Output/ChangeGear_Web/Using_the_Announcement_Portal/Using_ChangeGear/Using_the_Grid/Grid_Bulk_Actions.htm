Grid Bulk Actions

The grid views in each of the ChangeGear ticket modules (Incident, Change, Problem, Service Request,) also allow you to perform actions on more than one ticket in the grid. For example, you can highlight multiple Incident tickets listed in the Incident module grid and then close all of them with a single action instead of opening and closing each of them one at a time.

To perform a grid bulk action:

  1. Highlight one or more of the tickets in the grid by either holding down the shift key or clicking on multiple rows while holding down the control key.

  2. Click on the mouse right button. The following popup menu will appear.

Note: The maximum number of tickets that you can perform bulk actions on is 20.

The following actions can be performed on multiple grid tickets:

  • Promote: Any of the highlighted tickets in the grid that are in a promotable state can be promoted to new ticket s or all promoted to a specified existing ticket.

  • Close: Any of the highlighted tickets in the grid that are in a closeable state can be closed.

  • Assign: Any of the highlighted tickets in the grid that are in an assignable state can be assigned to a specified user/team/role.

  • Delete: All of the highlighted tickets in the grid will be deleted after a confirmation.

Bulk Action Results

After a bulk action is performed, a dialog will appear indicating how many of the tickets the action was successfully performed along with the IDs of the tickets that the action could not be performed on. An action may not be performed on a highlighted ticket if the ticket is not in a valid state to handle the action. For example, an Incident ticket that is not in a state that allows the Promote action will not be promoted during a Bulk Action promote.
