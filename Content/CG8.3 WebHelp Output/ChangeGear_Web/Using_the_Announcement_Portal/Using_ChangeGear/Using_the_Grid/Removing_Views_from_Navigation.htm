Removing Views from Navigation

You can remove your custom views you no longer wish to see in the Navigation Panel and still access the view in the View List menu.

To remove the view from the Navigation Panel:

  1. Select the view from the View List to display in the grid.

  2. Click Manage View in the grid tool bar.

  3. Select Save View Settings.

  4. Clear the check box Show in Navigation Panel.

  5. Click Save.
