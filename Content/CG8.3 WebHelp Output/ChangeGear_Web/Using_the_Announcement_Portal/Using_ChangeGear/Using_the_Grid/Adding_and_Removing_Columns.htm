Adding and Removing Columns

ChangeGear lets you drag and drop columns in and out of the grid. Columns that you drop onto the grid display data fields from the source records. ChangeGear hides columns that you drag out of the list. Later, if you decide that you need the columns again, you can drag them back onto the grid.

To add columns to the grid:

  1. In the Manage View menu on the grid toolbar, select Show/Hide Columns. Change Gear displays the Field Chooser.

  2. From the Field Chooser, drag the name of the column you want to add onto the grid near other column headers where you want the column to appear. ChangeGear adds the column to the grid.

To remove columns from the grid:

  1. In the Manage View menu on the grid toolbar, select Show/Hide Columns. Change Gear displays the Field Chooser.

  2. From the grid, drag the columns you want to remove from the heading back to the Field Chooser box. ChangeGear removes the column from the grid.
