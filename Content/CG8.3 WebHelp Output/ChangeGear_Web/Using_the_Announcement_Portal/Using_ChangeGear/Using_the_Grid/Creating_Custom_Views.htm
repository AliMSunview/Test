Creating Custom Views

In the grid, ChangeGear lets you sort and organize columns to customize your view of incident, problem, and change data. You can sort all of your data in ascending or descending order using the dynamic pyramid icon. To group your data, you can drag column headers into the column heading section.

Note: If the grid does not display the Group By box or the Filter Row, open your ChangeGear Preferences and modify your Grid Options.

To create a custom view:

  1. Add columns to the grid by dragging column headers to the grid from the column chooser. Remove columns by dragging the column header back to the column chooser.

  2. Sort the grid by clicking on a column (ID is shown selected in the graphic below) to sort your data based on that column. Observe a pyramid appears and the data is sorted using this column header.

  3. To group by a column header (Priority is being dragged and dropped in the graphic below), click and drag the header up into the gray column heading section.

  4. Filter the results by typing in a value in the filter boxes under the column headers, or by using one of the drop-down filters in the column header.

    Note: You can remove grid options that you no longer want to see, such as the Group by box or the filter boxes, by opening your ChangeGear Preferences, selecting Grid Options, and clearing the appropriate boxes. Removing the grid options does not remove your grouping or filtering.

  5. Save the View by opening the Manage View menu on the grid tool bar and selecting Save As New View. The Save View dialog displays.

  6. Enter the Name of the new view.

  7. Select Show in Navigation Panel if you want your new view to be visible in the Navigation Panel. Otherwise, the view is available in the View List located on the grid tool bar.
