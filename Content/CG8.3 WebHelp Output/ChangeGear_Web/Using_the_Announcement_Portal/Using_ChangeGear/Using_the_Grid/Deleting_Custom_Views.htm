Deleting Custom Views

If you no longer have a use for a view you have created, you can delete the view from both the Navigation Panel and the View List located in the tool bar. If you no longer wish to see the view in the Navigation Panel, you can edit the view so it is only be visible in the Manage View menu. See Removing Views from Navigation for more information.

To delete the view from the Navigation Panel and View List:

  1. Select the view so that it displays in the grid.

  2. Click Manage View in the grid tool bar.

  3. Select Delete Current View.

  4. Click OK in the confirmation dialog.
