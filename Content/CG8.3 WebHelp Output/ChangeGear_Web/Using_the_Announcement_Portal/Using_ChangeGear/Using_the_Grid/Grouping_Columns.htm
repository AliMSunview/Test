Grouping Columns

ChangeGear lets you group your items by column. For example, you can organize a list using the Priority column for grouping. The records are grouped according to their Priority, (e.g., Critical, High, Medium, Low). You can group by multiple columns, as in the image below.

After you group items, items display in a tree view with column entries used as the first level tree. If you place a second column on the group panel, the grouping displays as the next level in the tree. Click the arrow on the group by list to expand the grouping. Click the arrow again to collapse the grouping.

To group columns:

  1. In the grid, drag the column into the Group By box above the column header.

  2. Continue to drag columns into the Group By box.

  3. You can reorder the columns in the Group By box. See Drag and Drop Relationships for more information.

You can remove the grouping by dragging the column back to the grid header.

If the Group By box is not displayed in the grid, you can enable it by opening your ChangeGear Preferences.
