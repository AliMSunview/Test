Using the Grid

While working in ChangeGear, the section located beneath the tool bar is referred to as the grid, consisting of rows (records) and columns (fields). For example, the center section below provides a detailed list of the incident reports. You can use the View List menu to select what tickets appear in the grid and you can use the column headers to customize these tickets.

To view and work on a unique request, double-click on the item in the grid and the system opens the request in a separate window.
