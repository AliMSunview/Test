Using the Task Manager

You can view and edit your tasks from the Task Manager as well as from the ticket modules (e.g., Incident, Service Request, Change Request or Flex Request). Task Manager allows you to view all of your upcoming, current, and completed tasks. You can use the same grid options as the main ChangeGear grid to sort or filter items and use the Views menu to change how and which records are displayed, (e.g., my current tasks, my completed tasks).

The Task Manager is available from the ticket module navigation menu under the Other menu.

Note: If you cannot view the Task Manager, you may not have the correct privileges set to view it. Please contact your IT department if you feel this is in error.

To view or edit tasks:

  1. In the ticket module, open the Task Manager by clicking on the link in the group menu Other.

  2. Choose an appropriate view to easily find your task.

  3. Select the task.

  4. Click the task or the Edit button on the Task Manager toolbar to open the task.

  5. Make any changes.

  6. Click Save to save your change

    -OR-

    Click Cancel to discard any changes and return to the Task Manager.

Task Views

ChangeGear provides the following default views for tasks that you are assigned as the implementer of:

  • My Tasks: The default view. Displays your tasks that are in the new or in-progress state.

  • My Completed Tasks: Displays your tasks that are in the closed state.
