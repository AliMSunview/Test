Deleting Tasks

After you add tasks to a ticket, you may decide that one or more of the tasks are unnecessary.

To delete a task:

  1. Open the ticket.

  2. In the Implementation tab (RFC) or Tasks tab (Incident/Service Request/Flex), select the task you want to delete.

  3. In the tab toolbar, click Delete.

  4. Click Save.
