Updating Tasks

If you receive a task assignment, you can open the ticket and find it in the Implementation tab or in the Tasks tab. You can also find the task in the Task Manager, if you are able to access it. When you start the task, you should accept the task to indicate that you have begun your work.

The Implementation (RFC) or Tasks (Incident/Flex/Service Request) tabs displays the status of each task. The update lets the management team know that you have begun your task.

To update a task:

  1. Open the ticket.

    -OR-

    Open the Task Manager and open the task go to step 3.

  2. In the tab section, (Implementation tab for RFC or Tasks tab for other tickets), double-click the task.

  3. In the Notes box, type any information required to explain the implementation of the task.

  4. Click Save.
