Creating New Tasks

With each task, you and other assignees can provide a detailed account of your activities using the Implementation tab on the change request form or the Tasks tab for all other ticket forms.

Tip: You can quickly refer back to the parent ticket by clicking the ticket number in the Parent ID field.

To create a new task:

  1. Open the ticket.

  2. Click the Implementation tab (RFC) or Tasks tab (Incident/Service Request/Flex).

  3. In the tab toolbar, click New.

  4. In the Task, select the Implementer from the Implementer drop-down list.

  5. In the Description field, type a brief description of the new task.

  6. In the Type list, select the appropriate implementation task type.

  7. In the Start fields, select the start date and times of the implementation task.

  8. In the Completion fields, select the end date and times of the implementation task.

  9. In the Notes box, type any explanations or task requirements.

  10. Click Save.
