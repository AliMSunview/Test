Attachment URL

You can view the URL for an attachment. This feature is helpful when you want to imbed the URL in the actual text of a ticket.

To view an attachment URL:

  1. In the form, click the Attachments tab.

  2. In the Attachments list, select the attachment(s) you want to delete.

  3. In the Attachments toolbar, click Attachment URL.

  4. Save your changes to the form.
