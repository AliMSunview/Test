Adding Attachments

ChangeGear provides several ways to add attachments to your ticket in the RFC form. You can add attachments through the Analysis tab and the Release tab. The attachments you add through these tabs support requirements such as testing and training.

You can add attachments through the Attachments tab in all forms. These attachments can support anything you deem necessary for the ticket.

Tip: For instructions on adding attachments through the Analysis tab, see Attaching Requirement Documentation. For instructions on adding attachments to the QA tab, see Attaching Test-Related Documents.

To add an attachment:

  1. In the ticket form, click the Attachments tab.

  2. In the Attachments toolbar, click Add.

  3. In the Browse Attachments dialog box, click Browse and navigate to the appropriate directory location of the file you want to attach.

  4. Select the file that you want to add and click Open.

  5. In the Type list, select the type of file.

  6. Enter a descriptive name of the attachment. Entering a descriptive name allows users to quickly determine the purpose of the attachment, especially if the file name is too long to be easily viewed in the file name field.

  7. Click OK.

  8. You can verify your file is attached in the Attachments tab.
