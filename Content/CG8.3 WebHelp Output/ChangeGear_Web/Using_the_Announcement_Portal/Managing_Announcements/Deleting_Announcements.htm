Deleting Announcements

Refrain from deleting announcements as this negates electronic tracking. Instead, retract announcements. Deleting is a permanent change and cannot be undone. To delete announcements, you need announcement privileges.

Note: Deleting announcements that have already been sent will not retract e-mails, but will remove announcements from the ChangeGear Announcement Portal.

To delete an announcement using the Change module:

  1. Open the Change Management module.

  2. Open the RFC associated with the announcement you wish to delete.

  3. Open the Announcements tab.

  4. Select the Announcement you wish to delete

  5. Click Delete. A confirmation dialog box displays.

  6. Choose Yes to delete the announcement.

To delete an announcement using the Announcement Manager:

  1. Click Announcement Manager in the Other group menu. The Announcement Manager is displayed.

  2. Select the Announcement you wish to delete.

  3. Click Delete. A confirmation dialog box displays.

  4. Choose Yes to delete the announcement.
