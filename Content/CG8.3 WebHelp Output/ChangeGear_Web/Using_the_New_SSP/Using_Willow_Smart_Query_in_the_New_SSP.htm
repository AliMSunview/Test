Using Willow Smart Query in the New SSP

Smart Query is an advanced search that uses the Willow AI/machine learning technology built into ChangeGear. It lets you query the ChangeGear data source modules for which your company is licensed, and be presented with responses that could be from Knowledge Base articles, etc. 

Smart Query searches any module or module for which your company is licensed and enabled.

Smart Query is a separate page in ChangeGear that enables you to search for information independent of an incident. 

Smart Query is available to all ChangeGear users (Staff and End Users).

To access Smart Query:

  1. Click the Smart Query icon  in the upper right corner of your SSP page to open Smart Query.
  2. Enter your search term; as you key in text Smart Query looks for a similar topic and displays possible matches. Select the desired answer, or just hit Enter.

  1. Smart Query responds with available results displayed in tile form with summaries.

In this example, the first three responses fall under the Knowledge Base tab--you can click on any of those to get more details. Each response also shows a green bar if the result shows a high confidence in matching what you searched. There are also thumbs up (Helpful) and thumbs down (Not Helpful) icons that let you vote on how relevant you found the result. The vertical ellipses at the upper right provide More Options of Print and Share via email or Copy Link.

  1. To view more details on a particular response, click on its title. ChangeGear opens the corresponding slide-out. 

From this point, you can navigate to the next or previous item in the list by clicking the next or previous buttons. The tiles provide a summary of KB articles to easily identify if the article has the content for which you are looking.

  1. To move between tabs of response types, simply click on the type you want to view. For example, to use the incident responses, click Incident. ChangeGear then displays those results.

  1. From this point, you can navigate to the next or previous incident in the list.
