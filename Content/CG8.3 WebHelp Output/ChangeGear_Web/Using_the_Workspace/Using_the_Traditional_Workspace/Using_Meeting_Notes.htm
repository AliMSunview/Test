Using Meeting Notes

The Meeting Notes feature provides a centralized location for creating, storing, and sharing Change-related notes. Using meeting notes, you can create and share formatted notes without having to link them to a specific ticket.

Note: This feature is only available to users in the Change Staff persona. You may not have access to this ChangeGear feature depending on your permissions or ChangeGear edition.

Meeting notes are shared across all users in the Change Staff persona. However, you can share notes with other users by e-mailing the user with a direct link to the note.

To open meeting notes:

  1. Navigate to the Workspace from the ChangeGear primary navigation menu.

  2. Click the Meeting Notes icon on the right-hand side of the screen.

Navigating Meeting Notes

When the meeting notes feature is first opened, it shows a grid view of notes currently stored in ChangeGear. You can quickly preview a note by clicking on the note in the list. You can also print or search the contents of each note by using the preview window below the list.

To print a meeting note:

  1. From the grid, select the note that you wish to print.

  2. Click the Print button in the action menu. This opens your Web browser's print menu.

  3. Follow the prompts from your Web browser to print the note.

To e-mail a meeting note:

  1. From the grid, select the note that you wish to e-mail.

  2. Click the E-mail icon in the action menu. This creates a new e-mail using your device's e-mail client.

  3. Enter the recipient(s) you wish to send the note to and send the e-mail.

Creating and Editing Meeting Notes

The meeting notes editor lets you create, edit, and format meeting notes using a powerful HTML-based text editor. The editor provides word processing capabilities including font styling, font sizing, list creation, text alignment, table creation, link sharing, and spell checking.

To create a new meeting note:

  1. Click the Add  button to open the meeting note editor.

  2. Enter a Title in the title field.

  3. Enter the contents of the meeting note in the editor window.

  4. Click the Save button to save the meeting note. A notification dialog appears.

  5. Click the Close button to return to the grid view.

To edit a meeting note:

  1. If necessary, close any meeting notes that you have open and return to the grid view.

  2. In the grid view, Select the meeting note that you want to edit.

  3. Click the Edit  button.

  4. Make your changes, then click the Save button.

To delete a meeting note:

  1. From the grid view, select the meeting note that you wish to delete.

  2. Click the Delete button.

  3. Confirm the deletion by clicking OK.

Switching Between the Editor and Grid Views

In some cases, you may want to review a previous note while creating or editing another note. You can easily switch between the editor and grid views without having to close your current note.

To switch between the editor and grid views:

  1. Click the Add button to open the meeting note editor.

    - OR -

    Click on an existing Meeting Note and click the Edit button.

  2. Click the Back to Grid button to return to the grid view.

  3. In the grid view, click the Back to Meeting Note button to return to your note.
