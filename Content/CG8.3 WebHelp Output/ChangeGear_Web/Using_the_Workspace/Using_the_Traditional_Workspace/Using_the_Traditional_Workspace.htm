Using the Traditional Workspace

Access the traditional ChangeGear Workspace by opening the ChangeGear navigation menu and selecting Workspace.

Tip: If the SmartHub appears, click the < Workspace button in the top-right corner of the screen.

ChangeGear presents one of several preset Workspace layouts depending on your persona. Each user is also provided with a personalized Workspace called My Workspace, which you can customize with additional widgets similar to the Dashboard.

Tip: For more information on roles and personas, see Role vs. Persona under Using the Dashboard.

The centerpiece of the workspace is the Activities widget. The Activities widget is a grid displaying incidents, problems, changes, service requests, tasks, and flex tickets currently assigned to you. You can perform the same actions through the Activities widget as the item's respective module. This includes creating new items, deleting existing items, performing bulk actions, and sorting or categorizing the view.

The Workspace also provides global search functionality that returns results from multiple modules. For more information, see Advanced Searching.

Workspace Widgets

The Workspace provides several widgets:

  • My Views: The My Views widget displays a list of links to the views in the specified module. Clicking on a link will open the view in its respective module. You can limit the scope of the list to My Views, My Team Views, or Global Views.

  • My Links: The My Links widget displays a list of five user-definable links. You can link to other locations in ChangeGear, or to external web sites and web services. Clicking on a link opens the link in a new window.

  • My Reports: The My Reports widget lets you generate reports directly from the Workspace. It displays the reports listed in the My Reports section of the Reports module.

  • Activities: The Activities widget is a grid view of tickets that you created, requested, own, or are assigned to, as well as tasks that you are the implementer of. The widget displays all items by default, but you can filter to a specific module by clicking on the name of the widget and selecting a module from the drop-down menu. You can perform most of the same actions in the Activities grid as you can in each item's respective module.

    Note: Only one Activities widget can be visible at a time.

  • Score Card: The Score Card widget displays a specific data point from a ChangeGear module, such as the number of new incidents. The Score Card also uses data collected over time to display trends. For more information, see the Date Range section below.

For more information on using these widgets, see Customizing the Workspace.

Date Range

You can apply a specific date range to your Workspace widgets by selecting from the date range drop-down in the Workspace menu bar. For example, selecting Last 6 Months will refresh your Workspace widgets with data collected within the last 6 months.

Note: For the Workspace, this only impacts the Score Card widget.

The score card widget shows a trend arrow and percentage representing the amount of change between the current time frame and the previous time frame of the same duration. For example, after setting date range set to Today, a score card widget calculates the trend by comparing today's value with yesterday's value. Setting the date range to Current Month compares this month's results with last month's results. The percentage of change is calculated by taking the current count, subtracting the previous count, and then dividing the result by the previous count. The trend arrow shows an up arrow for a positive trend and a down arrow for a negative trend.
