Exporting Table Report Results

You can export your table reports to several file formats from ChangeGear.

Note: You can export table reports, other reports such as pie chart or bar chart cannot be exported.

To export your report results:

  1. Open the report you wish to export.

  2. In the Report Toolbar, use the menu to select the format.

  3. Click the Export button.
