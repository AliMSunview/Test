Overview of the Dashboard

The Dashboard provides an in-depth overview of ChangeGear for staff users and administrators. It presents Key Performance Indicators (KPIs) in the form of configurable widgets. Using the Dashboard lets you easily monitor the status of your ChangeGear modules from a single unified screen.

The Dashboard provides multiple configurable widgets including graphs, charts, gauges, and widgets that support custom embedded HTML. Each widgets (excluding the custom HTML widget) is configured using simple drop-down lists. You can specify the date range shown in each widget by using the Dashboard date slider or by selecting a pre-configured date range.

ChangeGear provides default Dashboards for factory personas. ChangeGear also supports team-specific dashboards, which can be created and modified by an administrator. Staff users can also create and modify their own personal dashboards.
