Knowledge Base Module

The Knowledge Base module, available only via the Web, displays knowledge base (KB) articles entered into the system. Only KB articles marked as public and published can be seen by end users. It provides all the tools needed to create and edit KB articles.

Depending on your role in ChangeGear or the version of ChangeGear you purchased, you may not have access to all the modules shown.

You can use the Knowledge Base module to:

  • Create, modify and delete KB articles

  • E-mail and print KB articles

  • Manage the KB article life cycle
