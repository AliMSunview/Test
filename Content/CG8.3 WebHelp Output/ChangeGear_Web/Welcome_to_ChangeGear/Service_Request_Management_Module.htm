Service Request Management Module

The Service Request module, available only via the Web, displays service requests entered into the system that you have permission to access. The Service Request module provides all the tools necessary to create and process service requests.

You can use the service request module to:

  • Create, modify and delete service request requests

  • E-mail and print service request requests

  • Manage the service request life cycle
