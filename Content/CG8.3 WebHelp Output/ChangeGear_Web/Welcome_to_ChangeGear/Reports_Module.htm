Reports Module

The Reports module lets you view and print reports and graphs. The left side of this module displays the report categories from which you can select specific reports. When you run a report, ChangeGear retrieves data from the database and displays the report in a separate window.

The Reports module in the ChangeGear desktop client also allows your ChangeGear administrator to create new reports that show up in the My Reports category as well as edit existing reports. In addition, the desktop client allows your ChangeGear administrator to set up schedules to generate and distribute reports.
