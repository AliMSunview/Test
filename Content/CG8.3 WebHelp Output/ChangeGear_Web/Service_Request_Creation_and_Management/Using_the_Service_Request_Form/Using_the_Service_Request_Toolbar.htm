Using the Service Request Toolbar

The Service Request Toolbar lets you complete tasks throughout the service request management life cycle. The toolbar provides command buttons in the form of icons.

Note: Some of these icons are only available when you are performing unique processes in the life cycle.

Icon

Description

Saves the service request.

Cancels changes made to a service request and closes the service request form.

Creates a new service request. When used within the service request form, it creates a new service request in the same window.

Creates a duplicate service request. When used within the service request form, it creates a duplicate service request in the same window.

Prints the service request.

E-mails the service request.

Displays a link to the service request.

Opens a dialog that allows you to search for other tickets or articles by module. You can also add a ticket or article from this dialog as a related item.

Submits the ticket to ChangeGear and begins the service request life cycle.

Accepts the service request.

Promotes the service request to an Incident, Problem or RFC ticket.

Sets the status to pending confirmation.

Validate fulfillment of a service request.

Closes the service request.

Reopens the service request.

Either creates a template from the current ticket by using "Save As" or overwrites the current ticket with the selected template. A drop-down option to create a ticket from a template is available after you select the service request type and the associated form displays. See Creating and Using Templates for more information.
