Submitting Service Requests

The service request (SR) life cycle begins with a submitted SR. You can create and submit a service request using ChangeGear's SR form on the Web, or by e-mailing an address set up by your ChangeGear administrator. You can select the type of service request and the appropriate form is displayed with the data fields necessary to fulfill the request. Specific fields are required before a service request can be submitted or saved as a draft.

Tip: For instructions on submitting service requests through e-mail, see Submitting a Ticket via E-mail.

You can elect to save the service request and return to it later without submitting the request. This scenario is referred to as the Unsubmitted work flow stage. Begin by creating a new service request and entering as much information as possible, click the Save button in the request toolbar. You must click Submit in the toolbar to enter the service request in ChangeGear. If you edit a ticket without saving, you are prompted to save when you close the ticket.

Tip: You can create a new ticket within the form. The new ticket appears in the same web browser as the service request you are editing. If you have any unsaved changes, you must save before creating a new ticket.

Note: The History tab on the form displays a history of changes made to the service request. These records provide a useful audit trail.

To submit a service request:

  1. Navigate to the service request module and click New in the toolbar. Select the request type from the list displayed.

    –OR–

    From the Self-Service Portal, click the Request Service button. Select a request type from the dialog window.

  2. Enter the required fields, (red fields with asterisk) on the Service Request form and press Save located on the toolbar.

  3. If you are ready to submit the request, press the Submit button located on the toolbar -or- you can return to the request later and submit.

  4. Click Submit.

To submit service request for approval:

  1. Double-click on the service request to open.

  2. Review the information in the service request. In the toolbar, click Submit for Approval.

  3. The service request enters the Approval State.
