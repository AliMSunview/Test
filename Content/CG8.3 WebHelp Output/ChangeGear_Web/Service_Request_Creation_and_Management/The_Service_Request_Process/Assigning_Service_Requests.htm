Assigning Service Requests

Once a service request (SR) is approved, ChangeGear automatically changes the stage of the request from Pending Approval to Assigned. You can assign the request to a user, team or role at any stage after the request has been created. The assignee accepts the request to start the fulfillment process.

To assign a service request:

  1. Click the service request to open.

  2. Enter an assignee in the Assign To field. You can click the ellipsis button and use the tabs to select a user, team or role, press OK.

  3. In the Service Request toolbar, click Save.

  4. In the Comments field of the Comment Popup dialog, type any comments you have.

  5. Click OK to save your changes.

To accept a service request:

  1. Click the service request to open.

  2. Review the service request data fields, click Accept in the toolbar.

  3. In the Comments field of the Comment Popup dialog, type any comments you have.

  4. Click OK to save your changes.
