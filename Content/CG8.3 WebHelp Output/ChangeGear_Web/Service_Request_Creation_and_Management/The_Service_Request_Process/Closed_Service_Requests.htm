Closed Service Requests

A closed service request requires no further consideration and its history is complete. However, you can reopen and reassign a Closed service request. You can type comment details in the comment dialog and reassign the service request to another assignee.

To reopen a closed service request:

  1. Click the service request to reopen.

  2. In the toolbar, click Reopen.

  3. In the comment dialog, type an explanation for reopening the service request.

  4. In the Assigned To field, click the user icon and select the user from the drop-down menu.

  5. Click Submit in the toolbar.
