Grouping Approvers

If you have sufficient privileges, you may be able to group approvers so that multiple people can receive notification at the same time. By default, approvers receive ticket notification in sequence, (i.e., approver order): Approver 1 receives notification of the ticket. After approval by Approver 1, Approver 2 receives notification, and so on. However, you can group two or more approvers to receive notification simultaneously.

You can determine whether a ticket has a grouping of approvers, by look at the order number next to each approver on the list. When multiple users have the same number, those users are part of a group.

The Conditions button allows you to decide whether every user in a group must approve the ticket or anyone in a particular group can approve the ticket.

To group approvers:

  1. Select the approvers you want to group together by holding down the Ctrl key and clicking each of the desired entries.

  2. Click Group. The approvers you grouped together have the same Order number; they will receive notifications simultaneously.

To set condition for approval:

  1. Press The Conditions button in the toolbar.

  2. To compel all users in the group to approve the ticket, check the Every user with the same ordinal must approve box. Uncheck the box, if any user in the group is sufficient to approve the ticket.

To ungroup approvers:

  1. Select the approver that you wish to remove from the group.

  2. Click Ungroup.
