Using the Grid

While working in ChangeGear, the section located beneath the toolbar is referred to as the grid. You can use the My Views group menu to select which requests appear in the grid and you can use the columns to further customize how the requests are displayed.

To view or update a request, double-click on the request in the grid and it will open up in a separate window.

To sort the grid:

  1. Click a column header (e.g., Item ID or Priority). Columns are initially sorted in ascending order.

  2. Click the column header again to sort the column in descending order.

To move columns:

Drag the column header to where you want the column to appear.
