Icon Library

The Icon Library provides a dedicated list of icons from which you can choose and apply when associating icons to items within Change Gear (widgets, Service items, Assets, etc.).

  1. Click the Icons button to open the Icons Library.
  2. Scroll to review and find the icon you want to select.

Note: You can click the More button to display additional icons.


3.  Once selected , click Apply to add it to the designated item.
