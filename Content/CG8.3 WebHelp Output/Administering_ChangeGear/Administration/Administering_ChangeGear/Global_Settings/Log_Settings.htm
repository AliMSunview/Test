Log Settings

The Log Settings section lets you configure the amount of detail captured in the ChangeGear logs. You can define the server log level as well as the client log level (per user and machine).

You can also use the Log Settings to direct ChangeGear errors to your in-house support team instead of the Serviceaide Software support staff.

Important: We suggest not performing any bug/debug procedures unless specifically recommended by one of our Serviceaide Software technical support representatives.

To define the log settings:

  1. Click the Administration tab.

  2. In the Administration section, click Global Settings.

  3. Scroll down to the Log Settings section.

  4. Click on the Server Log Level drop-down and select an option from the list.

  5. At the Client Log Level list, click the drop-down and select an option.

  6. If you want to send an e-mail with the server log and the client log details to support at Serviceaide Software, click Access Logs... to view the zipped log files.

  7. Click Save to keep your configuration changes.

Log Levels

Use the following list to decide how much detail is captured in the log.

  • Off - no detail is captured

  • Info - system activity/information is written to the log as well as errors

  • Debug - debug information is written to the log as well as system activity/information and errors

  • All - captures all detail available

  • Error - only errors are written to the log
