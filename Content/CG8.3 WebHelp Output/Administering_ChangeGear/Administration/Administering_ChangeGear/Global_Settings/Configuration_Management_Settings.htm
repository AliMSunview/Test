Configuration Management Settings

Often, the people who run the scans will not have access to all of your network. If your network requires certain credentials to access or scan different domains or devices, you can save them in the Configuration Management Settings. You can save different credentials per domain and add SNMP credentials. The passwords are encrypted and saved in the Credential Manager.

To add or edit domain credentials:

  1. Click the Administration tab.

  2. In the Administration section, click Global Settings.

  3. Scroll down to the Configuration Management Settings.

  4. Click on the Credentials button. The Credential Manager dialogCredential Manager dialog will appear.

  5. By default, the Credential Manager will open to the Domains tab. Click Add to add a new domain and credentials. If a domain has already been entered, you can edit it by clicking the Edit button. The Domain Properties dialogDomain Properties dialog will open.

  6. Enter the domain information and any credentials necessary for scanning the domain. Use the button to help you select the correctly formatted domain or user ID.

    Note: If you entered user credentials, you can validate the credentials you entered by clicking the Test Connection button.

  7. Check that the domain is enabled. If the domain is not enabled, the credentials will not be used to scan the domain, which can result in failed scans.

  8. Click OK.

  9. Click Save to keep your configuration changes.

To add or edit SNMP credentials:

  1. In the Global Settings Tasks group menu, locate Configuration Management Settings.

  2. Click on the Credentials button. The Credential Manager dialog will appear.

  3. Click on the SNMP tab to add or edit community strings.

  4. Click Add or Edit.

  5. In the Community String dialogCommunity String dialog, enter the description and community string.

  6. Check that the community string is enabled. If the domain is not enabled, the credentials will not be used to scan the domain, which can result in failed scans.

  7. Click OK to save your changes.

  8. Click Save to keep your configuration changes.

To add or edit SSH/Telnet credentials:

  1. In the Global Settings Tasks group menu, locate Configuration Management Settings.

  2. Click on the Credentials button. The Credential Manager dialog will appear.

  3. Click on the SSH/Telnet tab to add or edit credentials.

  4. Click Add or Edit.

  5. In the Range Properties dialogRange Properties dialog, enter the description and IP range.

  6. Click the Credentials button. The IP Credentials dialogIP Credentials dialog will display.

  7. Enter any credentials needed for scanning devices in the IP range.

    Note: You can validate the credentials you entered by clicking the Test Connection button. See IP Connection Test for more information.

  8. Click OK.

  9. Make sure the Enabled box is checked and click OK.

  10. Click Save to keep your configuration changes.
