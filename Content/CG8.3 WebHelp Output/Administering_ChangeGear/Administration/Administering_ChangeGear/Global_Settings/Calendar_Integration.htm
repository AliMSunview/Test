Calendar Integration

The Calendar Integration section lets you integrate ChangeGear with Microsoft Exchange, allowing users to access their Outlook meetings, reminders, and tasks in ChangeGear. Calendar integration allows ChangeGear to:

  • Access public and shared user calendars

  • Determine whether a user is out of the office

  • Create notifications for upcoming appointments and events

To enable calendar integration:

  1. Click the Administration tab.

  2. In the Administration section, click Global Settings.

  3. Scroll down to the Calendar Integration section.

  4. Click the Integration Enabled check box to enable calendar integration.

  5. Enter the URL of the Exchange Web Service in the Web Services URL field.

  6. In the Account Name field, enter the username of the account that ChangeGear will use to access items in Exchange.

    Note: Users will have to delegate access to this account in order to see their items in ChangeGear. Use an account that has–at a minimum–permission to read Out of Office status of other users.

  7. Enter the password to the account in the Password field.

  8. Optionally, click the Test button to check whether ChangeGear can access the account.

  9. Click Save to apply your changes.
