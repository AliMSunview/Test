Global Settings

The Administration module provides a global settings section that lets you configure the appearance and behavior of many ChangeGear features. Here you can perform actions such as customizing field colors, enabling/disabling automatic registration of users roles, configuring e-mail notifications, defining hourly rates, defining the detail level and history of logs, configuring e-mail notifications, and more.

  • General Settings - customize the field color for the required fields in forms

  • Web Application Settings - customize the location and appearance of the Web application

  • Security Settings - enable or disable automatic registration for user account and role creation and authentication

  • Configuration Management Settings - configure credentials to use when scanning CIs

  • Change Management Settings - define the hourly rate for every task in the RFC form's Implementation tab

  • Announcement Settings - define default recipients and how announcements are created

  • Notification Settings - configure e-mail notification

  • Knowledge Base Settings - configure how users interact with Knowledge Base articles

  • SLA Settings - enable or disable Service Level Agreements (SLA)

  • Log Settings - configure the amount of detail captured in the logs

  • Error Reporting Service Settings - configure how error reports are sent as well as view error reports

  • Business Hours - define your organization's business hour and holiday schedule

  • Restarting ChangeGear - restart ChangeGear and all of its dependencies

  • Repository Maintenance - configure how long to keep certain history records

  • LDAP Authentication Settings - configure LDAP authentication

  • E-mail Connector Settings - configure the E-mail Connector

  • Notification Email Header Settings - configure the master header used for all communication templates

  • Calendar Integration - connect ChangeGear to a Microsoft Exchange server to retrieve calendar events and reminders
