Announcement Settings

The Announcement Settings section allows you to both control how announcements are created and to specify a default recipient.

To customize the Announcement creation settings:

  1. Click the Administration tab.

  2. In the Administration section, click Global Settings.

  3. Scroll down to the Announcement Settings section.

  4. Select the "Suppress Creation Prompt" check box.

  5. Select your default creation method:

    • Copy from selected RFC - copies the Subject and Description from the RFC to the announcement.

    • Use selected template - applies the selected template to the announcement.

  7. If you choose "Use selected template," select a template. Templates are drawn from the Communication Templates sections.

  8. Click Save.

  9. Click the refresh button on the toolbar.

To add a default recipient:

  1. In the Announcement Settings section, enter an SMTP e-mail address in the Default Recipient field.

  2. Click Save.

  3. Click the refresh button on the toolbar.

Note: You can add multiple recipients by using a semi-colon between addresses.
