Change Management Settings

The Change Management Settings section lets you specify default settings used in the Change Management module. You can specify the rate that ChangeGear uses to calculate each user's hourly cost, as well as the default Change Mode users see when submitting a new change request.

Note: Both the hourly rate and default Change Mode can be configured on a per-user basis. To set a different hourly rate for the user, navigate to the Users page and enter the rate into the user's profileuser's profile. Users can set their default Change Mode themselves through the User Profile window in the ChangeGear Web client.

Hourly Labor Cost

The hourly labor cost is used to calculate the total cost of a change request. When a change request task is created, ChangeGear multiplies the number of hours needed to complete the task (as indicated by the Effort field) by the rate defined below. Refer to the RFC Creation and Management section of the ChangeGear User's Guide for more information on managing change requests.

Note: The rate entered in the Hourly Rate field serves as the default rate. You can specify different rates for individual users through the Users page of the Administration module.

To define the hourly labor cost:

  1. Click the Administration tab.

  2. In the Administration section, click Global Settings.

  3. Scroll down to the Change Management Settings section and click in the Hourly Rate text box.

  4. Type the cost per hour in the text box.

  5. Click Save to keep your configuration changes.

Change Mode

The Change Mode setting specifies the default form that end users see when they create and modify change requests. Users can also set their own preferred Change Mode through the User Profile in the ChangeGear Web client. The user setting overrides the global setting.

Tip: For more information on configuring Change Modes, see the Change Modes section under Configuring Lookup Lists.

To set the default Change Mode:

  1. Scroll down to the Change Management Settings section.

  2. If necessary, check the Enable Change Mode check box.

  3. Select the Change Mode in the Default Change Mode drop-down box.

  4. Click Save to keep your configuration changes.
