Configuring MAPI

The E-mail Connector supports the Messaging Application Programming Interface (MAPI). MAPI is a Microsoft-developed interface that allows Windows applications (such as Outlook, Word, and Excel) to create and send e-mails.

Before you can use MAPI with the E-mail Connector, you must first:

  1. Install Microsoft Outlook on the server hosting the E-mail Connector service

  2. Configure Outlook to use the same account as the E-mail Connector service.

To configure the E-mail Connector for MAPI:

  1. Log in to the server running the E-mail Connector service.

  2. Install Outlook on the server. The Outlook account must have the following characteristics:

    • The account must be a ChangeGear user with enough privileges to perform necessary actions such as create, edit, approve tickets.

    • The account must have local login privileges on the machine that runs E-mail Connector.

    • The account must be able to use the MS Outlook client on the ChangeGear server.

    • The e-mail address for incoming E-mail Connector messages must be associated with this account.

  3. Open the ChangeGear Desktop Client.

  4. Click the Administration tab.

  5. In the Administration section, click Global Settings.

  6. Scroll down to the E-mail Connector Settings section.

  7. Select the check box to enable the E-mail Connector.

  8. Select MAPI from the Connect using drop-down.

  9. Click Save to keep your configuration changes.
