Error Reporting Service Settings

When an error occurs, ChangeGear automatically collects information that can aid Serviceaide Software's staff in diagnosing the issue. The Error Reporting Service collects the important details—which usually includes log files—along with a brief summary of the configuration, and prepares the information to send it to Serviceaide Software.

You can choose to have all error reports sent to Serviceaide Software automatically as they occur, or you can manually send the error reports. If you would like to examine the type of information sent, you can view the details of an error report.

To select how error reports are sent:

  1. Click the Administration tab.

  2. In the Administration section, click Global Settings.

  3. Scroll down to the Error Reporting Service section.

  4. Select either:

    • Send Reports Automatically. Reports will be sent to Serviceaide Software as the occur, with no input required.

    • Send Reports Manually. You will need to use the View and Send Reports... button to open the ChangeGear Error Reporting Service dialog to send the reports.

  5. Click Save to keep your configuration changes.

To view error reports:

  1. In the Error Reporting Service Section, click the View and Send Reports... button.

  2. Select an error report.

  3. Click Show Details. The details pane will appear. You can navigate between the Summary, Stack Trace, and Server Log tabs.
