Managing Knowledge Base Articles

The Knowledge Base (KB) module allows users to capture system critical information, share and distribute knowledge, and provide self-guided troubleshooting for end users. The Knowledge Base module consists of individual articles that are created, reviewed, published, and distributed through ChangeGear Web. The Administration module lets you configure how staff users create and manage KB articles.

Note: For more information on using the Knowledge Base module, please refer to the ChangeGear User's Guide.
