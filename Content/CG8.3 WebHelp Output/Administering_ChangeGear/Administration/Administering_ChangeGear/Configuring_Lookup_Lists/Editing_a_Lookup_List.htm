Editing a Lookup List Item

To edit a lookup list item:

  1. Click the Administration tab.

  2. In the Administration module, select the lookup list to which you would like to edit an item. For example, if you wish to edit a company, you would select General > Companies, but if you wanted to edit an attachment category in the Problem Management module, you would navigate to Problem Management > Lookup Lists > Attachment Category.

  3. Select the item you want to edit.

  4. In the Common Tasks menu, click Edit.

  5. In the dialog, edit the details of the lookup list item.

  6. Click OK.
