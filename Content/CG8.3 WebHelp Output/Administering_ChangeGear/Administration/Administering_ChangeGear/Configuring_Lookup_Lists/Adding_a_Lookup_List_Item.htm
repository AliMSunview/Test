Adding a Lookup List Item

The procedure for creating lookup list items - such as Departments, Locations, Companies, etc. - are similar for all lookup lists in ChangeGear. You add to general lookup lists, such as companies or contacts, in the same way as you would add a new item in the Priority lookup list in the Change Management module or a new item in the Attachment Category for the CMDB.

You can edit lookup lists in the General section and individual management nodes under Lookup Lists.

To create a lookup list item:

  1. Click the Administration tab.

  2. In the Administration module, select the lookup list to which you would like to add a new item. For example, if you wish to add a company, you would select General > Companies, but if you wanted to add a new attachment category to the Problem Management module, you would navigate to Problem Management > Lookup Lists > Attachment Category.

  3. In the Common Tasks menu, click New.

  4. In the dialog, enter the details of the lookup list item.

  5. Click OK.
