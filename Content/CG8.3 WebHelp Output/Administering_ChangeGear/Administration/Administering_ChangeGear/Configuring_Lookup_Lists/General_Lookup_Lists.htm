General Lookup Lists

The GeneralGeneral section in the Administration tab lets you edit and delete lookup lists that are not associated with a particular module. Lookup lists contain the items used to populate menu fields throughout ChangeGear. The General section lets you define important criteria such as departments, locations, companies, and contacts.

Click for an example of the LocationLocation lookup list being accessed in an RFC form.

When creating user details for a user, click the Location drop-down and view a list of Location options for your change request.

You can define the menu options that appear in this drop-down list from the General > Location page in the Administration module.

Using the Common Tasks Menu

When customizing your lookup lists, you will be using the Common Tasks menu to create, edit, and delete your lookup list items. Click for details on using the Common Tasks menu.

The General Lookup Lists table below provides details on lookup lists and the relationship to fields in ChangeGear.

Lookup List

Function

Where the Lookup List is Used

Identifies departments within your organization.

CMDB > New > Department

Holiday Schedules

Identifies days in an operating calendar that are excluded from time tracking.

Web Client > SLA Manager

Integration Sources

Specifies integrations with third party issue tracking software. For more information, see Integrating with Third Party Issue Tracking Software under Advanced Customizations.

Administration > Module > Workflows / Business Policy Automation > Actions > Automation > Run Integration

Identifies your organization’s locations by city.

Administration > Security > Users > New > Create User Details > Location

CMDB > New > Location

Operating Calendars

Identifies business hours used for time tracking.

Web Client > SLA Manager

Resources

Identifies the resource. (Change or Suite only)

Resources > Resource Details > Resource

Resource Types

Identifies the types of resources that your organization manages which may feel the impact of an RFC. (Change or Suite only)
