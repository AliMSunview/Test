Deleting a Lookup List Item

To delete a lookup list item:

  1. Click the Administration tab.

  2. In the Administration module, select which lookup list item you would like to delete. For example, if you wish to delete a company, you would select General > Companies, but if you wanted to delete an attachment category in the Problem Management module, you would navigate to Problem Management >Lookup Lists > Attachment Category.

  3. Select the item you want to delete.

  4. In the Common Tasks menu, click Delete.

  5. In the delete confirmation window, click OK.
