Lookup Lists

ChangeGear uses customized and predefined lists of items to populate the dialog boxes, drop-down lists, menus, etc. throughout the application. You can add, modify, and delete items from the lookup lists to better support your organization's structure. The items you add to the Administration module's lookup lists configure all of the other ChangeGear modules and dialog boxes. For example, you might add a priority to the Priorities lookup list. This item will now be available to you whenever you select a new priority in the form.

Important: After modifying a lookup list, restart the Internet Information Services (IIS) process or recycle the CGWeb application pool to refresh the ChangeGear Web client.

The GeneralGeneral node in the Administration module lets you configure companies, contacts, departments, and locations while the management nodes allow you to configure lookup lists specific to that node. See General Lookup Lists for more information.
