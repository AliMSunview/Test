Using the Label Editor

You can modify any label used in ChangeGear to better suit how your organization uses the product. A label is a method of describing entity attributes, displayed as a field label or column header. The Label Editor is used to modify default labels, labels across modules, and labels on column headers. For example, if you use the Priority field to indicate the customer type or tier, you could change the field name to be "Customer Type." The Customize Labels utility is used to change individual labels across the product.

ChangeGear includes three different methods to change labels.

Note: Methods are listed in order of precedence, i.e., changes made to the model will supersede changes made via the Label Editor, which in turn takes precedence over the Form Designer. Not all editions of ChangeGear have access to all of the methods.

To change the labels via the Label Editor:

  1. In the Administration module, select Advanced.

  2. Select Customize Labels. The Edit Labels dialog will display.

  3. In the Find what field, enter the string you wish to change.

  4. Enter the replacement text in the Replace with field.

    Note: You can leave the Replace with field blank if you wish to edit the text manually.

  5. Click Find Next to find the text you wish to replace. The Strings tree will open to the text, while the Path field will show exactly where to view the string.

  6. Click Replace to replace the text, or click Edit to replace the text manually.

  7. Continue replacing text until you have modified all the relevant strings.

  8. Click OK when you are done.
