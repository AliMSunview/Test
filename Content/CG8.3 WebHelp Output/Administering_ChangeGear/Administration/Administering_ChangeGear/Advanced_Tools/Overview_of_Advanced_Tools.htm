Advanced Tools

ChangeGear allows you to tailor the underlying entities, create custom forms, modify form fields and labels, and add new Lookup Lists. Modifications are then available throughout ChangeGear. For example, you can create a new form to be used for service requests that only has fields relevant to service requests and new fields not available in the default ChangeGear tickets, such as a new drop-down list for service request type.

Note: You may not have access to this ChangeGear feature depending on your permissions or ChangeGear edition.

Important: Serviceaide Software highly recommends using a test system and creating a backup of your ChangeGear database and installation folder before using the Entity Editor.

Using the Advanced tools, you can

  • Modify labels on forms

  • Customize forms

  • Create new fields and lookup lists to be used in forms

  • Save customizations

  • Create new forms to be used with different types of requests

  • Create new CI types (see Extending the CMDB for more information)

There are three tools in the Advanced section, depending on your edition of ChangeGear.

The Entity Editor – ChangeGear provides an extensible entity model with which you can add entities or tailor existing entities, their fields, forms, and associations. Entities in ChangeGear include RFCs, Incidents, Users, and Tasks. The Entity Editor also allows you to edit labels, similar to the Label Editor (See Using the Label Editor for more information). To begin editing entities, see Using the Entity Editor.

The Label Editor – Use the Label Editor to modify labels throughout ChangeGear. Labels in ChangeGear are the text associated with a field and are often found above or to the left of a field. Labels are also used as column headers. Labels are also known as captions in the Entity Editor. See Using the Label Editor for more information.

Export Tool – Exports the Customization folder and Form Layouts, Views, Roles, Workflows, and any Entity Model changes. See Exporting Customizations for more information.
