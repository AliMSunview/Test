Exporting Customizations

Use the Export Tool to save modifications made to your ChangeGear installation to a single, easily transferred ZIP file. The tool exports the Customization and Views folders and includes any modified or new forms, views, workflows, roles, and any changes to the entity model.

You can use the ZIP file to facilitate:

  • Parallel production environments

  • Perfecting customizations in test environments

  • troubleshooting changes with Serviceaide Software Support

To export customizations:

  1. In the Administration module, select Advanced.

  2. Select Export Tool.

  3. Select a location to save the Customizations.zip file.

    Note: You should not store the Customizations.zip file in the ChangeGear folder or subfolders, as this area may be deleted when upgrading.

  4. Save the ZIP file.

To import customizations:

  1. In Windows Explorer, navigate to your ChangeGear server folder. By default it is located at C:\Program Files\SunView Software\ChangeGear\Server.

  2. Unzip your customization folder in the ChangeGear Server folder.

  3. Allow the files to be overwritten.

  4. Restart the ChangeGear service.

  5. Recycle the ChangeGear Web application.

Your imported customizations will now be available in ChangeGear.
