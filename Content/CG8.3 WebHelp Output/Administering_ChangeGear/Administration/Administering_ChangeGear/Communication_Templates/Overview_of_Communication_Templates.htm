Communication Templates

Communication templates are customizable templates that define the appearance and layout of ChangeGear notifications. In addition to providing a consistent look to all notifications, templates make it easy to embed information about ChangeGear items–such as tickets, announcements, and Knowledge Base articles–into outgoing e-mails and SMS notifications. Communication templates are configured separately for each module.

Note: For instructions on how to add a global header to all templates, refer to the Configuring Notification Settings section under Global Settings.

To create a Communication Template:

  1. In the Administration tree, select Communication Templates under the module you wish to modify.

  2. From the Common Tasks menu, click New. The Communication Template Details dialog displays.

  3. In the Communication Template Details dialog, type a name in the Name field, a description in the Description field, and a subject in the Subject line.

  4. If you would like to use another template as the basis for this template, click Import and choose the template you want to import.

  5. Enter the contents of the template in the body area. Use the toolbars to customize the style, format, and layout of the template.

  6. If you want to quickly add fields to your text, click on the Click here to select a field... button to select from a wide variety of fields you can insert into your template. For information on unique or unlisted fields, see Special Fields.

  7. Select a field and click Insert.

  8. Optionally, click the Test Send button to send a test e-mail so you can verify the style and format.

  9. Click OK to save the template.
