Special Fields

The Communication Template Details Dialog supports several special fields which are not listed in the Fields drop-down. These fields are listed below along with their use.

Note: These fields may depend on certain modules or features to be activated before they can be used.

Module Field Description
Generic $(WebURL) Displays the ChangeGear Web client URL.

Tip: You can change this field by modifying the Web URL field in the Global Settings.
$(ID) Displays the unique ID number of the ticket.
$(MostRecentComment) Displays the most recent comment added to the ticket.
$(Description[10]) DIsplays the ten most recent entries in the ticket's Description field.
$(Description[10]:ET) Displays the ten (10) most recent entries in the ticket's description field with the entry dates formatted in Eastern time.
$(Description[0].Comment) Returns the value of the comment field for the first entry in the Description field.
$(Module) Displays the name of the module that the item belongs to.
SmartResponder

Note: The Intelligence Pack must be installed and enabled for these fields to work. For more information, see the Customizing Web Settings topic under Global Settings.

 $(SmartResponder_SuggestedKBs) Displays up to two (2) of the most relevant Knowledge Base articles as suggested by the machine learning service.
$(SmartResponder_ChatBotLink) Opens a new session with Willow, ChangeGear's automated chat agent, for further troubleshooting a ticket.

E-mail Connector

Note: The E-mail Connector must be installed and enabled for these fields to work. For more information, see the Configuring the E-mail Connector topic under Global Settings.

 $(CG_MID) Returns an encrypted copy of the ticket ID. This is used internally by ChangeGear to identify the ticket that the e-mail is referring to.
$(CG_ACTIONS) Displays a list of ticket actions in an e-mail as text. Users can perform an action by replying to the e-mail and placing an "X" in between the brackets next to an action.
