Configuring Notification Rules

Notification Rules can be configured in one of two locations:

  1. In the main Administration tree under each module's respective Notification Rules node

  2. In the Workflow Editor by adding a Notify via Email automation action

Either method allows you to customize the appearance and behavior of notification rules based on the action performed. These rules can be configured independently for e-mail and for SMS.

To configure notification rules:

  1. Click the Administration tab.

  2. In the Administration menu, navigate to the management module that you wish to configure notification rules for and click the Notification Rules node.

  3. Notification rules are independently configurable for all enabled workflows. Click the arrow icons next to the workflow that you wish to modify to view the available rules.

  4. Select the Action that you wish to modify and click Edit to open the Notification Rule Details window. This window lets you customize the contents of the notification, as well as its intended recipients.

  5. If necessary, enable the notification rule by checking the Enabled text box.

  6. Click the Edit button to open the Communication Template Details window. For e-mails, this appears next to the Subject text box. For SMS messages, this appears next to the Template text box. This window lets you customize the layout and appearance of the message. For more information, see the Communication Templates chapter.

  7. Add or remove recipients by using the Add... and Remove buttons next to the Recipients list box. For more information, see the Adding and Removing Recipients topic.

  8. Click OK to save your changes.
