Creating Roles

ChangeGear comes with a predefined set of service desk roles. You can, however, modify the descriptions and privileges of these roles to meet your organizational requirements. You can also add new roles if you don't find anything that applies to your organization’s roles.

Tip: The roles listed in the Administration module populate the Roles field in the Users Details dialog box.

Note: Site License users (End Users or Requesters) cannot be assigned to a custom role.

To create a Role:

  1. In the Administration explorer, navigate to Security > Roles.

  2. In the Tasks group menu, click New.

  3. In the Role Details dialog, General tab, type your role in the Name field.

  4. In the Description field, type a brief description of the role.

  5. The Persona field directly relates to defined layouts for dashboard and workspace when the role has access privileges to these functions. When persona is blank, the system defaults the dashboard and workspace layouts to the IT Staff published layout. In addition to accessing the role's persona layout, the user can also personalize My Workspace or My Dashboard layouts to suit their needs.

    Refer to the User's Guide, Using the Dashboard and Using the Workspace sections for a complete explanation of layouts (personas).

  6. You can edit role membership, define role privileges, or define field security at this time.

  7. Click OK to create the new role.
