Editing Role Membership

As the responsibilities of your staff shift, you can add or remove members from a role. ChangeGear assigns users in these roles to specialized change management tasks. Tasks are permitted according to the privileges defined for that role. For instance, members of the IT Staff role receive implementation assignments.

Important: You must acquire a named seat license from ChangeGear to perform any functions in ChangeGear other than those allowed by the Requester role.

To update Role membership:

  1. Click the Administration tab.

  2. In the Administration module, navigate to Security > Roles.

  3. Select the role that requires the membership change.

  4. In the Common Tasks menu, click Edit.

  5. In the Roles Details dialog box, click the Members tab.

  6. Determine if you want to add or remove a member.

    • If you want to add a member, go to Step 7.

    • If you want to remove a member, go to Step 10.

  7. Click Add.

  8. In the Select User dialog box, type or select the user that you want to add to the role and click OK.

  9. Confirm the user is added to the Members tab and go to Step 12.

  10. Select the user you want to remove from the Members tab and click Remove.

  11. Confirm the user is removed.

  12. Click OK to save you changes.
