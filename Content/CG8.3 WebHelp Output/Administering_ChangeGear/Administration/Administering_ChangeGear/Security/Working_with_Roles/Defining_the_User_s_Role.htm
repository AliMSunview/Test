Defining the User's Role

When you are creating or editing Users, you can define the role requester when completing the User Profile.

To define the Role for the User:

  1. Click Administration > Security > Users.

  2. In the Common Tasks menu, click New to create a new user and define the role

    - OR -

    Select an existing user and click Edit.

  3. In the User Profile section, click the Role drop-down menu.

  4. Select the role from the list.

  5. Complete the remaining fields in the dialog.

  6. Click OK.
