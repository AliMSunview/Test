Managing Users

The Administration module lets you manage the creating, editing, and removal of users and roles in ChangeGear. From the Users page you can manage your users, and you can define a variety of important user details.

You can manage your user roles from the Roles page to create and edit roles.

You can manage teams from the Teams page to create and edit teams.
