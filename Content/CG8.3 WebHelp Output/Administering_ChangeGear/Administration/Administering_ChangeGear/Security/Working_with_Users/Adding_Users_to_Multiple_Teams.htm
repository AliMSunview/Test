Adding Users to Multiple Teams

You can quickly add users to multiple teams in the Teams tab of the User Details dialog. The Select Team dialog allows you to multi-select teams using the SHIFT and CTRL keys.

Note: When adding multiple security teams at one time, the user's role for all the selected security teams will be the same. You can change the user's role in the team in the User Details dialog after the user has been added to the team.

Also note that the roles available to a team member also depend on the user's license type.

Security teams may not be available in your version of ChangeGear.

To add users to multiple teams:

  1. Navigate to Administration > Security > Users.

  2. Select the user you wish to add to multiple teams.

  3. Navigate to the Teams tab.

  4. Select Add to Team. The Select Team dialog will display.

  5. Select the team(s).

  6. Click OK.

Note: If you have selected a security team, you will be prompted to select a role for the user.

The user is now added to the selected teams. You can change the user's role within a security team or remove the user from a team directly from the User Details dialog.
