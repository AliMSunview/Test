Specifying a User as VIP

A VIP (Very Important Person) user is a user that you wish to distinguish from other users as well as possibly provide additional privileges or services.

For ChangeGear staff users (not end users), ChangeGear will highlight VIP users with a VIP icon next to their names as well as highlighting any tickets they request with a VIP icon. Depending on your business requirements you can also customize ChangeGear to use the User VIP state and/or the ticket VIP state to provide additional VIP specific functionality.

To specify a User as VIP:

  1. Click Administration > Security > Users.

  2. Select the appropriate user.

  3. In the Common Tasks menu, click Edit.

  4. In the User Details dialog, select the VIP check box.

  5. Click OK to save your changes.
