Enabling SMS Notifications

ChangeGear allows you to send notifications to users via Short Message Service (SMS). Before users can begin receiving SMS messages, they need to provide their mobile phone number as well as opt-in to receiving SMS messages. You also need to configure ChangeGear to connect to your SMS provider. For more information on configuring SMS, see Configuring Notification Settings under Global Settings.

To enable SMS notifications for a User:

  1. Click Administration > Security > Users.

  2. Select the user you wish to modify.

  3. In the Common Tasks menu, click Edit.

  4. In the User Details dialog, enter the user's mobile number in the Mobile text box.

    Note: Please check with your SMS provider for the required format of this field.

  5. Select the SMS Opt-In check box. Alternatively, end users and staff users can opt-in to receive SMS notifications through their User Profile in the ChangeGear Web client.

  6. Click OK to save your changes.
