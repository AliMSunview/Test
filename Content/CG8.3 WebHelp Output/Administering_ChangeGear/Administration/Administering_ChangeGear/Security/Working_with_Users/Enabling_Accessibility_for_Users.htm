Enabling Accessibility for Users

In compliance with Section 508, SunView Software provides an accessibility mode for those users who interact with computers via assistive technology. The accessibility mode enhances the usability of ChangeGear for screen readers, enabling users to interact with ChangeGear entirely from the keyboard.

Users will be able to open tickets from the grid, navigate between tabs, and sort column headers.

To enable accessibility:

  1. Click Administration > Security > Users.

  2. Select the appropriate user.

  3. In the Common Tasks menu, click Edit.

  4. In the User Details dialog, select the Accessibility Enabled checkbox.

  5. Click OK.

When the user next logs in to ChangeGear, the user will have access to the accessibility options. Users can learn more about accessibility in the Enabling Accessibility topic of the ChangeGear User's Guide.
