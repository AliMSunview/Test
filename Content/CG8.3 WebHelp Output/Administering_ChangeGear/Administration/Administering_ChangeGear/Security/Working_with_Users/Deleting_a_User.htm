Deleting a User

From the User page in the Administration module you can delete a ChangeGear user.

To delete a User:

  1. Navigate to Administration > Security > Users.

  2. Select the user you want to delete.

  3. In the Common Tasks menu, click Delete.

  4. In the Confirm Delete window, click Yes.

Note: Although you deleted this user from ChangeGear, their name will still appear on any past tickets or CIs with which they are associated. We recommend that you do not permanently purge users from the database.
