Creating an Account for External Users

Many organizations have users that do not have access to the corporate network, but still need to access ChangeGear. Examples include those that rarely use the corporate network, and therefore have a laptop with only a workgroup account, or external users that may need to submit service or change requests. ChangeGear's integrated authentication cannot be used to authenticate such users. Instead, they must be authenticated by ChangeGear Authentication, or by configuring ChangeGear to connect to an LDAP server. See Configuring LDAP Authentication for more information.

You can create ChangeGear specific credentials using the ChangeGear Authentication settings on the User Details dialog without relying on your LDAP or AD system to authenticate the users. The user will be able to log on to both ChangeGear Web and the desktop client.

To add an external User:

  1. Navigate to Administration > Security > Users.

  2. In the Common Tasks menu, click New.

  3. Enter the user name you are assigning to the external user.

  4. Choose ChangeGear Authentication.

  5. Enter and confirm a password.

  6. Choose a license type.

  7. Fill out the User Profile.

  8. Make sure the account is enabled.

  9. Click OK.

Note: ChangeGear Authenticated users can change or reset their passwords using ChangeGear Web.
