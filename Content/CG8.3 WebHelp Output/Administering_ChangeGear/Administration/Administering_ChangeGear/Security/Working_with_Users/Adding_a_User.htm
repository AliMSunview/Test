Adding a User

You can add a user to ChangeGear from the UsersUsers page and define user details such as the appropriate license for the user, the logon security access, the type of authentication to be used, and user profile information. To save time and increase security, some details are automatically populated from Windows AD.

Integrating Windows AD

A key feature of ChangeGear is the integration of user authentication using Microsoft Windows Active Directory (AD), or your network domain controller. Using AD, users don't have to re-enter their credentials when accessing ChangeGear. This feature saves time, credentials are easier to manage because there are no separate user passwords to maintain, and if a user's Windows network account is disabled their ChangeGear account becomes disabled also.

When creating users in ChangeGear, some key details are automatically populated from the AD. For example, Windows may provide the user’s first name, last name, and e-mail address. You can also select to allow users to automatically register (auto-registration).

View Security Settings in the Global Settings section for more information about enabling auto-registration.

ChangeGear also supports LDAP integration. View Configuring LDAP Authentication for more information.

If you are manually entering user names (for example, creating ChangeGear authorized users), you can use the person's e-mail address as their user name. ChangeGear allows alpha-numeric characters and the following special characters to be used for user names: . @ ! # $ % & ' * + - / = ? ^ _ ` { | } ~

Additionally, user names may not be more than 256 characters long and IDs may not be more than 100 characters long.

To add a User:

  1. Navigate to Administration > Security > Users.

  2. In the Common Tasks menu, click New.

  3. In the User Details dialogUser Details dialog, click Search by the User Name field if the user exists in your Windows network. If the user does not, choose an alternative authentication type and skip to step 6.

  4. Enter the users by:

    • Selecting User Type and Location

    • Typing the user name

    • Clicking the Check Names button

    - OR -

    • Clicking Advanced

    • Clicking Find Now.

    • Selecting the user from the list. If the user is not found, confirm that the location indicated in the From this location field is correct

  5. Click OK.

  6. In the Authentication section, choose how you want the user to be authenticated.

    • If you choose LDAP Authentication, click Search. LDAP Authentication must be enabled for this selection to be active.

    • If you choose ChangeGear Authentication, manually enter the User Name and Password.

  7. Choose the license the user will use.

  8. In the User Profile section, complete the user detailed information fields.

    Tip: Click for more details on selecting the Department, Location, Manager, Role, and User Hourly Rate.

  9. Make sure the account is enabled.

  10. Click OK.

Additionally, you may enable accessibility, select a proxy approver, specify a user as a VIP, add the user to teams, or enable SMS messages for the user.

Note: You do not have to complete every field in the Users Details dialog box, but you must complete the Role field. ChangeGear’s assignments and privileges depend on the roles of your users.
