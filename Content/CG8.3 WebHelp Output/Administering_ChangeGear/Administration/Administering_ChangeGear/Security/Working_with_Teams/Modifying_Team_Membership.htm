Modifying Team Membership

You can select the team members either after a team is created or during the creation. Members can be added or deleted as well as have their role within the team modified to adjust for differing responsibilities or employment shifts.

Note: New or modified team members must refresh their ChangeGear client or re-login to ChangeGear to view changes to their roles within the team.

Note: Teams may not be members of other teams.

To modify Team membership:

  1. Click the Administration tab.

  2. In the Administration module, navigate to Security > Teams.

  3. Select the team that requires the change.

  4. In the Common Tasks menu, click Edit.

  5. In the Team Details dialog box, click the Members tab.

  6. Determine if you want to add or remove a member.

    • If you want to add a member, go to Step 7.

    • If you want to remove a member, go to Step 10.

  7. Click Add.

  8. In the Select User dialog box, type or select the user that you want to add to the team and click OK.

  9. Confirm the user is added to the Members tab and go to Step 12.

  10. Select the user you want to remove from the Members tab and click Remove.

  11. Confirm the user is removed.

  12. Click OK to save your changes.

To modify the Role of a Team member:

  1. Click the Administration tab.

  2. In the Administration module, navigate to Security > Teams.

  3. Select the security team that requires the change.

  4. In the Common Tasks menu, click Edit.

  5. In the Team Details dialog box, click the Members tab.

  6. Select the member that you wish to modify.

  7. Click Edit.

  8. In the Select a Role dialog box, choose what role the user will have.

  9. Confirm that the user's role has been successfully changed.

  10. Click OK to save your changes.
