Creating and Editing Views

You can create or edit views that can then be accessed by all or a subset of users using the Administration module. Users can create their own custom views in their client, but those views will only be visible to the user that created them.

Note: With the proper privileges, ChangeGear allows users to manage views, (i.e., create and edit), directly in the Web using the Web View Editor. Please refer to the Web Help Administration section, View Management for an explanation of this function.

To create or edit views:

  1. In the Administration explorer, navigate to the management node of your choice and choose Views. For example, if you wanted to modify a view in the Incident Module, you would navigate to Incident Management > Views.

  2. In the Tasks group menu, click New. If you are editing a view, select the view and click Edit.

  3. Follow the instructions on the Tasks group menu of the View Designer (located on the left):

    • Choose which columns you want in the view.

    • Group or Sort the columns.

    • Filter the results.

  4. Click Save. The View Details dialog displays.

  5. In the Name field, type the name of your custom view. You will not be able to edit the Name field if you are modifying a view.

  6. In the Description field, type a short description of the view.

  7. If you do not want the view deployed at this time, do not select Visible - If unchecked, users will not be shown this view.

  8. If you want to the view to be visible on the Navigation Panel of ChangeGear Web in addition to the Views drop-down menu, select Appears in the Navigation Panel of ChangeGear Web.

  9. If you'd like to use the Criteria Editor to filter your report results, click on the Filter tab and add filter criteria.

  10. If you wish to specify how ownership and team membership impact what is viewed, click the Ownership tab and select from the following:

    • All Viewable - All users can see the view, but the items displayed by the view may be restricted based on owners, teams, and privileges.

    • User's Team - If a user is part of a team, the user will be able to select this view and the view will display items the team owns.

    • User's Own - This view will always be visible to the user and displays only tickets that the user owns.

  11. Click OK. ChangeGear will display a prompt when the view is saved.

  12. Close the View designer.
