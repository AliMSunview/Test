Using the Criteria Editor

Use the Criteria Editor to further constrain the number of viewable items. For example, if you wish to only view items that have been closed in the last week, you can use the Criteria Editor to do so.

Tip: You can create views that will only be triggered by logged in users. Select a user field and then choose "The User" or "The User's Team."

The Criteria Editor allows you to create criteria in either a basic mode where you select fields and rules with pull down menus or in and advanced mode which allows you to enter your criteria in an SQL-like format. It is recommended that you add the fields you want with the basic editor and then switch to the advanced editor to fine tune your query.

Note: If you switch from advanced editor back to the Basic editor you will be warned since doing so will clear out any query you had put together with the Advanced criteria editor.

To specify basic filters:

  1. Open the Criteria Editor. You can do so from the Reports or Views dialog by selecting Save or by clicking the link under Step 3 in the instructions.

  2. Move to the Filter tab.

  3. Make sure the Basic sub tab is highlighted

  4. Choose a field to filter on.

  5. Choose an operator (For help with list selections, see Advanced Search Operators).

  6. Choose the value that you wish to filter on.

  7. To specify more criteria, choose AND or OR at the end of the row. You can delete conditions by clicking on the red X at the end of the row.

    Note: You can select multiple fields from the same aggregate category, i.e., you can select both the Assigned To and Location fields from the General section.

  8. Click OK. If you have opened the Criteria Editor by clicking on the Save button, you will save the report or view.

To specify advanced filters:

  1. Open the Criteria Editor. You can do so from the Reports or Views dialog by selecting Save or by clicking the link under Step 3 in the instructions.

  2. Move to the Filter tab.

  3. Make sure the Advanced sub tab is highlighted.

  4. Type in the query you wish in the text field.

  5. Validate the syntax of the query by clicking on the Validate button.

  6. Click OK. If you have opened the Criteria Editor by clicking on the Save button, you will save the report or view.
