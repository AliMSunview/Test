Run Code

Run Code allows you to execute custom C# code within ChangeGear. Run Code is not exposed by default and must be enabled in the Desktop Client configuration.

Warning: Adding a Run Code automation allows you to run unfiltered C# code on the ChangeGear server. Please verify the code on a test system first, as it may cause system instability or data loss.

To expose the Run Code automation:

  1. On the ChangeGear server, open the Desktop Client configuration file (defaults to C:\Program Files (x86)\SunView Software\ChangeGear\Client\CG.exe.config).

  2. Search for the EnableRunCodeAction key, which is found in the appSettings section.

  3. Change the value of the EnableRunCodeAction key from false to true.

    <add key="EnableRunCodeAction" value="true"/>

  4. Save and close the file.

  5. Restart the Desktop Client.

Now when you add an automation in a BPA or workflow, Run Code will appear in the Action Type drop-down.

To learn more about creating and editing BPAs, see the Business Policy Automation chapter of the ChangeGear Administrator's Guide.

Using Run Code

Before you can create a Run Code action, you first need to know the exact syntax of the code you wish to run. This will vary based on the task you wish to perform, the module that the BPA is in, and several other factors. For more information, please contact SunView Software Support at (813) 632-3600 or at support@sunviewsoftware.com.

To create a Run Code action:

  1. In the Name field, enter the name of the new action.

  2. Optionally, enter a Description of the new action.

  3. In the Code field, enter the C# code that will execute when the action is performed. Remember that this has the potential to cause instability or data loss if the command is entered incorrectly. Please confirm with a SunView Software Support specialist before executing the action.

  4. Make sure the Enable check box is checked, then click OK to save the action.
