Announce

When you select Announce in the Action Type list, the Run Script/Program Action Details dialog box displays.

  1. In the Name field, enter the name of the Action.

  2. In the Description field, click and type a description of the Action.

  3. In the Status drop-down menu, select the state that you want the created Announcement to be.

    Tip: Select Published if you want the Announcement directly published to the Announcement Portal.

  4. Click Edit to create the announcement.

    Note: To create an announcement with the same subject and body as the RFC, use the variables $(Parent.Summary) in the Subject field and $(Parent.Description) in the Body field.

  5. Click Add to enter recipients if you want to e-mail the announcement. If you do not want to e-mail the announcement, leave this field blank.

    Note: Make sure the action is enabled.

  6. Click OK.
