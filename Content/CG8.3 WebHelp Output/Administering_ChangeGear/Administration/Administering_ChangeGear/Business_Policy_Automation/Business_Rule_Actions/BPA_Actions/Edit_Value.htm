Edit Value

When you select Edit Value in the Action Type list, the Edit Value Details dialog box displays.

Note: Make sure the action is enabled.

  1. In the Name field, enter a name for the action.

  2. In the Description field, type a description of the action.

  3. In the Field Name field, select the name of the field you want to modify. This will determine which values can be entered in the Value field.

  4. In the Value field, first select what type of value you will be entering.

    • Literal: Select a value that already exists in ChangeGear or enter a specific date.

    • Relative: Enter a value relative to when the BPA runs ("Now"). This is only available with Date/Time fields. You can enter any number of hours between 0 and 86700.

    • Expression: Enter an SQL expression and/or a ChangeGear substitution variable to calculate the value. Use the test button to check the validity of the expression or variable.

  5. Click OK.

Note: The history tab will reflect changes made by BPAs.
