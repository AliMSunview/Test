Determining Approvers

If you have created multiple approver groups, only one group is applied per ticket per state. ChangeGear determines which approval group to apply to a ticket, based on the following hierarchical list.

  1. Approval Groups specified in the Workflow Editor.

  2. Approval Groups based on criteria selected in the Approval Group Details dialog.

  3. Approval Groups based on ticket Type selected in the Approval Group Details dialog. For example, when approval group is for a service request ticket, Type can be hardware request, software request, etc.

  4. Default Approval Group.
