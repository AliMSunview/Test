Editing Approval Groups

The Approval Groups feature allows you control which users approve Change or Service Request tickets and when the approvers receive notifications. ChangeGear comes with a default Approval Group that you can modify to further streamline your organization’s ticket management work flows.

Note: This feature is supported by the Change Management, Service Request Management, and Flex modules.

To modify an Approval Group:

  1. Click Administration tab.

  2. In the Administration module, navigate to Change or Service Request module, e.g., Change Management > Approval Groups. The Approval Groups page displays.

  3. Select the Approval Group you want to modify, and click Edit in the Common Tasks menu. The Approval Group Details dialog box displays.

  4. Enter your changes (for more information on Approval Group Details entries, see Creating Approval Groups).

  5. Click OK to save.

Pre-authorized Users

The Pre-Authorized tab lets you define a list of users whose change requests do not need to be approved by an approver. Pre-authorization creates a fast-track for trusted users to automatically move their change requests into the next phase of the change request life cycle. As such, pre-authorization takes precedence over the list of approvers.

Important: Pre-authorization is determined by the user who moved the ticket into the Approval state, not necessarily the user who submitted the ticket.

When a pre-authorized user moves a ticket into the Approval state, the ticket is automatically approved and moves into the next state. ChangeGear also updates the ticket history to reflect that the ticket was approved by a pre-authorized user.

To add a user to the list of pre-authorized users:

  1. Open the Approval Group Details dialog box for the Approval Group that you want to modify.

  2. Click the Pre-Authorized tab to display the list of pre-authorized users.

  3. Click Add to open the Select User dialog. Select the user, team, role, or dynamic role that will be pre-authorized for this particular group.

  4. Click OK to save your changes.
