How Do I Create Forms to Be Used With Specific Resource Types?

You can create forms to be used when new CIs are created with specific resource types, including sub-resource types. If you do not create or modify a form for a new CI entity, then the Base CI's active form will be used.

Tip: If you add fields before you save a new CI entity, the Configuration tab is automatically populated with the new fields.

To create forms:

  1. In the Administration module, select AdvancedAdvanced.

  2. Click Entity Editor in the Customization page. The Entity EditorEntity Editor will display.

  3. Select the CI entity in which you wish to create a form. For example, Configuration Management > CIs> Storage > Forms.

    Note: If you want to create custom forms for factory default CIs, you may need to create new forms in the Base CI and then specify the form to use in Configuration Management > Lookup Lists > Resource Types.

  4. Click either Edit Form or select the Default form and click the Customize button.

  5. Customize your form as necessary. See Customizing Forms for more information.

  6. Save your changes in File > Save All Changes.
