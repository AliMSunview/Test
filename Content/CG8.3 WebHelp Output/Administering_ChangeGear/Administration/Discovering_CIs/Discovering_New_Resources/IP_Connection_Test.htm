IP Connection Test

The IP Connection Test window allows you to test SSH, Telnet, or administrator credentials against a specific computer. This allows you to verify that the credentials are correct without having to run a complete scan.

To test the credentials using a known IP address or host name:

  1. Enter the IP address or host name of the computer that you wish to connect to.

  2. Specify how long the test should wait for a response in seconds before ending.

  3. Click Test Connection. The results of the test will display in the text box.

  4. Click OK to close the window and return to the IP credentials window.
