Duplicating Items

If you receive many of the same types of tickets - for example, if a major server was down and you continuously received incident reports about it - or if you have recurring processes, such as routine maintenance requests, you may wish to choose a ticket and duplicate it for every call you receive. You can change the information in the duplicated item, such as the Requester and Due Date to match the new information.

You can duplicate incidents, problems, RFCs, and CIs in ChangeGear, though CIs can only be duplicated using the desktop client. In the desktop client you can use the shortcut menu, the edit menu, or the keyboard shortcut to duplicate an item.

Duplicated information includes attachments, business services, tasks, and approvers.

To duplicate an item:

  1. Select the item in the grid view.

  2. Right-click the item you wish to duplicate and select Duplicate

    -OR -

    Open the Edit menu and select Duplicate

    -OR-

    Select the item and press Ctrl+U.

  3. Make any necessary changes to the duplicated item.

  4. Save the changes.
