Adding Attachments

You can add attachments through the Attachments tab. These attachments can support anything you deem necessary for the resource.

Note: You may have an option to save a file as a link. This option should only be used with files that will not be moved and are saved to a networked storage device, such as a shared drive. This option does not attach the file to the CI. Instead, the 'attachment' acts like a link to the file, similar to links on Web pages.

To add an attachment using the Attachments tab:

  1. In the CI Editor, click the Attachments tab.

  2. In the Attachments toolbartoolbar, click New.

  3. In the Browse Attachments dialog box, click Browse and navigate to the appropriate directory location of the file you want to attach.

  4. Select the file that you want to add and click Open.

  5. In the Type list, select the type of file.

  6. Enter a descriptive name of the attachment. Entering a descriptive name allows users to quickly know the purpose of the attachment, especially if the file name is too long to be easily viewed in the file name field.

  7. Click OK.

  8. You can verify your file is attached in the Attachments tab.
