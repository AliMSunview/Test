Sorting the Grid

Sorting items in the grid helps to organize your resources in a meaningful way. You can also sort resources that have already been sorted without affecting the original list. For example, if you sorted items by type to more easily find your Web servers, you could also sort the items by location to see how your Web servers were distributed.

You can sort your columns in the following ways:

  • Click the column header. ChangeGear sorts the column in ascending order. A triangle will appear on the column header, pointing up.

  • In the grid, right-click the column header of the field you want to sort and select Sort Ascending.

  • Click the column header a second time. ChangeGear sorts the column in descending order. A triangle will appear on the column header, pointing down.

  • In the grid, right-click the column header of the field you want to sort and select Sort Descending.

  • Hold the SHIFT key down while clicking in a second column header will sort by a second field.

To remove sorting from your columns:

  • Hold the CTRL key down while clicking in a sorted column header removes sorting by that column and preserves other sorting conditions.

    - OR -

    In the grid, right-click the column header and select Clear Sorting.
