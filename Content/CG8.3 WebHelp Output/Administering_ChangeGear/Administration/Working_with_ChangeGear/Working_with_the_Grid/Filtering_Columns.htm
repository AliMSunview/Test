Filtering Columns

You can display and hide records in ChangeGear's grid by filtering columns. When you filter a column, the list displays only the records that meet certain conditions that you specify. Each column displays a set of standard values and field-specific values, based on the records currently in the list. You can also build a custom filter to apply the conditions. You can filter columns by using standard filters or by using the 'Custom' field in the column header.

To filter a column using standard filters:

  1. In the column header you want to filter, hover over the column head and click the key.

  2. Select a filter from the list.

    • All - displays every record in the grid

    • Custom - displays only the records that match your conditions for the field

    • Blanks- displays only the records that contain blank values in the field

    • Non blanks - displays only the records that contain a value in the field

    • Field names - displays only records with the same field name

     

To create a custom filter:

  1. In the column header, click the filter icon which appears when you mouse over the column header.

  2. Select Custom. ChangeGear displays the Custom AutoFilter dialogCustom AutoFilter dialog box.

  3. In the first drop-down list of the Custom AutoFilter dialog box, select the condition that you want to apply to the field.

  4. In the first text box, select or type the value of the condition.

  5. If you want to apply a second required condition, select the And radio button. If you want to apply a second optional condition, select the Or radio button.

  6. In the second text box, select or type the value of the condition.

  7. Click OK to apply the condition.

Filter Conditions

  • Equals - displays only the records that match the condition you choose or specify for the field

  • Does not equal - displays only the records that do not match the condition you choose or specify for the field

  • Is greater than - displays only the records that display a value greater than the one you choose or specify

  • Is greater than or equal to - displays only the records that display a value greater than or equal to the one you choose or specify

  • Is less than - displays only the records that display a value less than the one you choose or specify

  • Is less than or equal to - displays only the records that display a value less than or equal to the one you choose or specify

  • Blanks - displays only the records that display null values in the selected field

  • Non blanks - displays only the records that display a value in the selected field

  • Like - displays records that contain the condition you specify

  • Not like - displays records that do not contain the condition you specify
