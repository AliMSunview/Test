Drag and Drop Relationships

ChangeGear allows you to drag and drop columns both in the column header and in the group panel. This allows you to have the most relevant data displayed first.

To move column headers:

  1. Click and hold the mouse button on the column header you want to move.

  2. Drag, while still holding the mouse button down, the column header to the appropriate place.

  3. Release the mouse button.
