Grouping Columns

ChangeGear lets you group your items by column. For example, you can organize a list by the Type column. This groups records according to their type. You can group by multiple columns, as in the image below. After you group items, items display in a tree view with column entries used as the first level tree. If you place a second column on the group panel, the second level tree will be the second column entries.

To group columns:

  1. In the grid, drag the column into the group panel,

    - OR -

    In the grid, right-click the column headers and select Group By this Column.

    Note: If you do not see the group panel, click the icon in the toolbar, or go to the View panel and select "Group Panel."

  2. Continue to drag columns into the group panel.

  3. You can reorder the columns in the group panel. See Drag and Drop Relationships for more information.

To clear grouping:

  1. In the group panel, right-click.

  2. Select Clear Grouping.

This will remove all grouping you have done.
