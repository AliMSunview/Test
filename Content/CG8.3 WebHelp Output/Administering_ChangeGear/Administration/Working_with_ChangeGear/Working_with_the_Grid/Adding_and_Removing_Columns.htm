Adding and Removing Columns

ChangeGear lets you drag and drop columns in and out of the grid. Columns that you drop onto the grid display data fields from the source records. ChangeGear hides columns that you drag out of the list. Later, if you decide that you need the columns again, you can drag them back onto the grid.

To add columns:

  1. In the View menu, click Column Chooser. ChangeGear displays the Column Chooser,

    - OR -

    In the ChangeGear toolbar, click the Column Chooser icon ,

    - OR -

    In the grid, right-click the column headers and select Column Chooser.

    Note: If you are editing a view or a report, the Column Chooser automatically displays.

  2. From the Column ChooserColumn Chooser, drag the name of the column you want to add onto the grid near the other column headers. ChangeGear adds the column to the grid.

To remove columns:

  1. From the grid, drag the columns you want to remove back to the Column Chooser.

    - OR -

    Drag the column in the grid until a large black "X" appears, and then release the column. 

  2. ChangeGear removes the column from the grid.
