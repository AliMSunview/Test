ChangeGear Privileges

Each ChangeGear module has specific privileges associated with it. Privileges are modified via the Role Editor in the Administration module.

Note: This topic applies to staff roles. To see privileges available to requester roles, view Requesters Privileges.

Team

Privilege

Capabilities

Add

Allows a user to add a team using the ChangeGear Desktop Client under Security > Teams.

Delete

Allows a user to delete a team using the ChangeGear Desktop Client under Security > Teams.

Edit

Allows a user to edit team information using the Desktop Client application or by using the Team Manager on the web. Users with the Edit privilege enabled for the Team object in the Role Details dialog, can edit team information for any team.

Edit Own

Allows a user to edit team information for which the user is the team owner using the Desktop Client application or by using the Team Manager on the web. Users with the Edit privilege enabled for the Team selection in the Role Details dialog, can edit only teams which they own.

View

Allows a user to view teams using the ChangeGear Desktop Client under Security > Teams. The Team Manager module on the web is unavailable unless the user has the Edit privilege or Edit Own privilege enabled for the Team selection in the Role Details dialog. View does not impact the ability to edit, use, or view team owners as they pertain to incident, problem, change tickets or service requests.

Access Module

Allows a user to view the teams via the Team Manager module on the web ONLY when the user has the Edit privilege or Edit Own privilege enabled for the Team selection in the Role Details dialog. Access module does not impact the ability to edit, use, or view team owners as they pertain to incident, problem, or change tickets.
