List of Controls

ChangeGear provides a collection of customizable controls that you can add to forms, customize, and remove from forms.

Empty Space Item

Adds a control that modifies the appearance of a form by controlling the space between form elements. The minimum width of an empty space item is 10 pixels.

Label

Add a label to a group of controls.

Tip: Use the context menu to modify the caption. Simply right-click the control and select Change Caption.

Attachment Button

Add an attachment button to the form, like the attachment buttons on the Analysis tab.

Note: You need to know the Attachment table name and key column name before adding the control to the form, which can be found in the ChangeGear database.

Drop Down List

Create a non-editable, drop-down list populated by items stored in a database column to make selections instead of relying on user input. For example, Company can be moved to the form as a field or a drop-down list could be moved to the form that draws its data from the Company field. In the case of the Company being selected as a field, the user must correctly type the company name into the field. If Company was added as a drop-down list, the user only has to select the company name from the drop-down list.

Note: You need the DataMember, DataTextField, DataValueField, and FieldName information.

The DataMember refers to which entity/database table contains the information to populate the drop-down list. You can find the entity name by looking at the LookupType property of the field you are creating the drop-down list from or by viewing the database. The entity will be the last part of the string in LookupType, e.g., IOriginType. You must view the field properties in the Entity Editor and not the Form Designer. You can also use the ChangeGear database to lookup the appropriate table.

The DataTextField and DataValueField information can be found in the field properties of the Entity Editor.

Note: If you are drawing the information from a field you created, the DataMember will look similar to I_UDT_FieldName.

Property

Description

DataMember

The name of the entity that contains fields specified by DataValueField and DataTextField, e.g., IResourceClass or IDepartment, which can be found in the Entity Editor in the FieldDataType property of the field or the ChangeGear database.

DataTextField

The name of the field used for the display text of each drop-down item, e.g., Name or Department, which can be found in the Field Properties under FieldName.

DataValueField

The name of the field used for the value of each drop-down item, e.g., Name or Department, which can be found in the Field Properties under ID.

DefaultValue

The default value of the field.

DynamicFilter

The filter rules to apply on other controls. When a user selects an item, other fields on the form are filtered based on the rules defined in this property.

FieldName

The name of the field that is updated when the form is saved, e.g., ManagementStatus or Department, which can be found in field list.

Filter

The filter used as the criteria for drop-down items. Filters are in the format [Field] OPERATOR 'Filter'. For example [ImpactedManagedResourceTypeFullPath] LIKE 'Business Service%' will filter the list of impacted resources to only those with "Business Service" in their name.

The text that appears when the mouse cursor is hovering above the field.

The name of a ChangeGear view whose criteria is used to filter out items in the drop-down list, e.g. Teams.

Entity Picker

Create an editable, drop-down list populated by items stored in a database column to make selections instead of relying on user input. For example, Summary can be moved to the form as a field or a drop-down list could be moved to the form that draws its data from the Summary field. In the case of the Summary being selected as a field, the user must type the summary into the field. If Summary was added as a entity picker, the user can select an appropriate summary from the drop-down list.

Note: You need the DataMember, DataTextField, DataValueField, and FieldName information.

The DataMember refers to which entity/database table contains the information to populate the drop-down list. You can find the entity name by looking at the LookupType property of the field you are creating the drop-down list from or by viewing the database. The entity will be the last part of the string in LookupType, e.g., IOriginType. You must view the field properties in the Entity Editor and not the Form Designer. You can also use the ChangeGear database to lookup the appropriate table.

The DataTextField and DataValueField information can be found in the field properties of the Entity Editor.

Hyperlink

Use the Hyperlink control to add a link to a Web page to the form. The hyperlink can make use of ChangeGear's substitution variables.

Note: You need to know the URL and how you'd like the link to be displayed (i.e., if you wish to use a textual link or an image link).

Property

Description

Caption

The text that describes the field.

CaptionLocation

The location of the caption relative to the field.

ControlType

The type of control used for this field (non-editable).

HyperlinkText

The hyperlink text, e.g., Click Here.

ID

The unique ID of this field (non-editable).

ImageURL

The path of an image to display for the hyperlink. If this is set, HyperlinkText will not be displayed.

IsVisibleOnWeb

Indicates whether this item will be displayed on the form on the Web.

OnClickScript

The browser script that runs when the hyperlink is clicked. For example, if (confirm("Are you sure you want to search for '$(Summary)'")) {return true;} else {return false;} will display a pop-up window.

OtherProperties

Additional user specified properties. Properties must be in the format of [Property]="[Value]".

Note: This is an advanced configuration property; unless you are working with a Professional Services representative, you will likely never enter a value in this property.

ShowCaption

Indicates whether the caption is show for this field.

Target

The target window into which the web page specified by the URL is loaded. The standard selections - _blank, _parent, _self, and _top - are available.

URL

The path to the webpage, e.g., http://sunviewsoftware.com. You can use substitution variables with the URL, e.g., http://google.com/search?q=$(Summary).
