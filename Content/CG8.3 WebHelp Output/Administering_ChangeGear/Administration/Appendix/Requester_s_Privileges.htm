Requester's Privileges

The Requesters role is unique in that you can only remove the privileges for users with this role. This is the role that non-named seat licenses use.

Note: If you make use of security teams, any changes you make to the Requester roles may also impact your security team.

The following table describes the privileges and capabilities of the default Requester role.
