REST API Entities

This section lists the entities (and their properties) that can be accessed through the REST API. Entities are listed according to module, name, REST API name, description, and access permissions. To learn more about interacting with these entities, see the Entity topic in the Using the Rest API section.

Note: You may not have access to certain entities depending on your ChangeGear license.

Note: For Flex entities, replace the number after "Flex" with the desired Flex module number (for example, "Flex1Ticket" references a ticket in the Flex 1 module).
