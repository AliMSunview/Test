Overview of the ChangeGear Mobile App

The ChangeGear mobile app brings the functionality of the ChangeGear Web client to Android and iOS devices. With the ChangeGear mobile app, you can request help from your support team, review outstanding tickets, view and receive announcements, and browse Knowledge Base articles from your mobile device.

You can download and install the ChangeGear mobile app for free from your device's app store. For more information, please see the Installing the ChangeGear Mobile App section of the ChangeGear Installation Guide.
