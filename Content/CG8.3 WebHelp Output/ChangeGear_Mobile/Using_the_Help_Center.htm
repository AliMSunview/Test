Using the Help Center

The Help Center provides quick access to Help Requests, Service Requests, and Knowledge Base articles. From here you can submit new tickets, review open tickets, and find additional resources.

Note: Only end users can view this page.

The Help Center provides the following actions:

  • Request Service: The Request Service button opens a form for creating a new service request ticket. For more information, see Creating and Viewing Service Requests.

  • Request Help: The Request Help button opens a form for creating a new incident ticket. For more information, see Creating and Viewing Help Requests.

  • Service Requests: The Service Requests button displays your service request tickets that are in the New or In-Progress state.

  • Help Requests: The Help Requests button displays your incident tickets that are in the New or In-Progress state.

  • KB: The KB button opens a list of Knowledge Base articles. For more information, see Viewing Knowledge Base Articles.

  • Call Support: The Call Support button uses your mobile device's dialer app to call your support team. If your device does not have a dialer app installed, then the app displays the contact number for your support team. It also provides a link for you to add the number to your contacts list.
